## 1. Tom tat

Bai nay giai thich **Build portfolio projects** trong boi canh Data Analyst hien dai. Sau bai hoc, ban nen biet topic nay giup tra loi cau hoi data nao, nam o dau trong workflow va co the bien thanh query, spreadsheet, notebook, chart, dashboard hoac portfolio artifact.

## 2. Muc tieu hoc tap

- Giai thich duoc Build portfolio projects bang ngon ngu cua ban.
- Nhan biet topic nay nam o dau trong workflow data analyst.
- Ap dung vao mot dataset, query, spreadsheet, notebook, chart hoac dashboard nho.

## 3. Khai niem chinh

- Build portfolio projects giup bien ky nang thanh bang chung nang luc.
- Portfolio tot co cau hoi, dataset, cleaning steps, analysis, visualization va recommendation.
- Nen viet README ro va noi ket qua voi business decision.

## 4. Vi du / Demo

```text
README outline:
business question -> dataset -> cleaning -> analysis -> charts -> recommendations -> next steps
```

## 5. Bai tap thuc hanh

- Tao README project gom cau hoi, data source, method va ket qua.
- Them it nhat 2 chart va 3 recommendation.
- Ghi lai limitation va next step cho project.

## 6. Loi thuong gap

- Hoc thuoc dinh nghia nhung khong ap dung vao dataset.
- Khong ghi lai gia dinh va caveat.
- Tao output kho tai lap vi thao tac tay qua nhieu.

## 7. Checklist hoan thanh

- Toi co the giai thich **Build portfolio projects** trong 1-2 phut.
- Toi co mot vi du nho hoac artifact thuc hanh cho bai nay.
- Toi biet topic nay lien quan den dataset, metric, chart, insight hoac business question nao.
- Toi da ghi lai it nhat mot caveat hoac cau hoi can phan tich tiep.

## 8. Outcome lien quan

Turn skills into visible portfolio projects, Kaggle practice and a sustainable learning routine.

## 9. Tong ket

**Build portfolio projects** la mot moc trong lo trinh Data Analyst. Hay bien no thanh mot bang tinh, query, notebook, chart, dashboard hoac portfolio note de kien thuc co cho bam.
