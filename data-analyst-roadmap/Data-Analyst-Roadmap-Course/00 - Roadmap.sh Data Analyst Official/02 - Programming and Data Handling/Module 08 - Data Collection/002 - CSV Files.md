## 1. Tom tat

Bai nay giai thich **CSV Files** trong boi canh Data Analyst hien dai. Sau bai hoc, ban nen biet topic nay giup tra loi cau hoi data nao, nam o dau trong workflow va co the bien thanh query, spreadsheet, notebook, chart, dashboard hoac portfolio artifact.

## 2. Muc tieu hoc tap

- Giai thich duoc CSV Files bang ngon ngu cua ban.
- Nhan biet topic nay nam o dau trong workflow data analyst.
- Ap dung vao mot dataset, query, spreadsheet, notebook, chart hoac dashboard nho.

## 3. Khai niem chinh

- CSV la format trao doi data don gian nhung de loi encoding, delimiter va type.
- Can kiem tra header, separator, quote, date format va missing value marker.
- File analysis nen giu ban raw va ban cleaned rieng.

## 4. Vi du / Demo

```text
Raw data -> validation checks -> cleaning rules -> cleaned dataset -> data quality notes
```

## 5. Bai tap thuc hanh

- Chon mot dataset nho va ghi data source, schema, row count.
- Viet checklist quality: null, duplicate, type, range va outlier.
- Luu ban raw va ban cleaned voi changelog ngan.

## 6. Loi thuong gap

- Khong luu data raw nen khong tai lap duoc analysis.
- Xoa/sua du lieu ma khong ghi rule va ly do.
- Bo qua missing value, duplicate hoac data type sai.

## 7. Checklist hoan thanh

- Toi co the giai thich **CSV Files** trong 1-2 phut.
- Toi co mot vi du nho hoac artifact thuc hanh cho bai nay.
- Toi biet topic nay lien quan den dataset, metric, chart, insight hoac business question nao.
- Toi da ghi lai it nhat mot caveat hoac cau hoi can phan tich tiep.

## 8. Outcome lien quan

Collect data from databases, files, APIs and web pages with clear assumptions and data lineage.

## 9. Tong ket

**CSV Files** la mot moc trong lo trinh Data Analyst. Hay bien no thanh mot bang tinh, query, notebook, chart, dashboard hoac portfolio note de kien thuc co cho bam.
