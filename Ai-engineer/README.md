# AI Agent 365 Ngày

Ghi chú và tài nguyên từ khóa học AI Agent 365 ngày — hành trình từ cơ bản đến triển khai production với các mô hình ngôn ngữ lớn (LLM).

## Cấu trúc khóa học

### Tuần 1 — Giới thiệu & Cài đặt
- Tổng quan về LLM Engineering
- Cài đặt môi trường, Jupyter Notebook

### Tuần 2 — Gradio & Transformers
- Xây dựng giao diện với Gradio
- Làm việc với Hugging Face Transformers

### Tuần 3 — Mô hình mã nguồn mở
- Khai thác open-source models
- So sánh và lựa chọn mô hình phù hợp

### Tuần 4 — Đánh giá mô hình
- Kỹ thuật đánh giá (evaluation) LLM
- Benchmark và metrics

### Tuần 5 — RAG Pipeline
- Xây dựng Retrieval-Augmented Generation từ đầu
- Vector database và embedding

### Tuần 6 — Dữ liệu & Machine Learning
- Quản lý và xử lý dữ liệu
- Tích hợp ML truyền thống với LLM

### Tuần 7 — Fine-tuning & Phát triển chuyên nghiệp
- Fine-tuning mô hình
- Sinh code và text, tối ưu hóa edge model

### Tuần 8 — Triển khai Production
- Production deployment strategies
- Structured output, monitoring
