# Module 07 - Backup Recovery and Validation

## Ket qua dau ra

Design, run and validate logical and physical backup/restore strategies for PostgreSQL.

## Project

Lab: Back up and restore a database using pg_dump/pg_restore plus a base backup strategy document.
