# Module 06 - Security

**Hoc phan:** 02 - Installation Configuration and Security

## Ket qua dau ra

Secure PostgreSQL with roles, privileges, authentication, SSL, row-level security and OS-level controls.

## Project

Lab: Create least-privilege roles, configure pg_hba.conf rules and add RLS to a tenant table.
