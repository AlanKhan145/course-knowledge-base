# Module 03 - Types of Data Analysis - Các kiểu phân tích dữ liệu

## Mục tiêu module
Phân biệt descriptive, diagnostic, predictive và prescriptive analysis trong bài toán kinh doanh.

## Bài học
- [001 - Descriptive Analysis](001 - Descriptive Analysis.md)
- [002 - Diagnostic Analysis](002 - Diagnostic Analysis.md)
- [003 - Predictive Analysis](003 - Predictive Analysis.md)

## Thực hành trọng tâm
Lấy một metric bị giảm và viết 4 câu hỏi tương ứng với 4 kiểu phân tích dữ liệu.

## Output sau module
- Analysis type decision guide with business examples
- Ghi chú decision/report có thể chia sẻ với stakeholder
- Danh sách câu hỏi dữ liệu, rủi ro metric hoặc việc cần phân tích tiếp
