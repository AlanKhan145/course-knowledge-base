## 1. Tóm tắt
Nội dung này tập trung vào **Cloud-Native Game Backend** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Hoàn thành project **Cloud-Native Game Backend** thành artifact portfolio có thể chạy hoặc review.
- Ghi lại kiến trúc, protocol, database/cache/queue và cách test.
- Tạo README rõ ràng với scope, setup, demo và limitation.

## 3. Nội dung roadmap
- Containerize services
- Deploy lên Kubernetes
- Health check

- Load test

## 4. Bài tập thực hành

## 5. Artifact nên tạo
