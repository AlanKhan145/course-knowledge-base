## 1. Tóm tắt
Nội dung này tập trung vào **Game Backend Microservices** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Hoàn thành project **Game Backend Microservices** thành artifact portfolio có thể chạy hoặc review.
- Ghi lại kiến trúc, protocol, database/cache/queue và cách test.
- Tạo README rõ ràng với scope, setup, demo và limitation.

## 3. Nội dung roadmap
- Auth service
- Player profile service
- Inventory service
- Match service
- Leaderboard service
- PostgreSQL + Redis + Kafka
- Docker Compose deployment

## 4. Bài tập thực hành
- Xác định scope nhỏ, chạy được trong local hoặc Docker Compose.
- Viết README setup, architecture diagram và test scenario.
- Ghi lại limitation, future work và bài học production.

## 5. Artifact nên tạo
