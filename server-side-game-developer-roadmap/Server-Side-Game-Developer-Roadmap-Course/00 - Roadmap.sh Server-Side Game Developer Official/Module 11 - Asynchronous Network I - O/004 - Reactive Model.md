## 1. Tóm tắt
Nội dung này tập trung vào **Reactive Model** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Giải thích được **Reactive Model** trong luồng kết nối client-server của game online.
- Biết trade-off latency, reliability, ordering, throughput và security.
- Tạo được demo hoặc ghi chú packet/protocol để dùng khi xây server thật.
