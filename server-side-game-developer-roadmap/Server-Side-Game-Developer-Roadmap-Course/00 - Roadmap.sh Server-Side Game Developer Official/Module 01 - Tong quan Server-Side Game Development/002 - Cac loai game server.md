## 1. Tóm tắt
Nội dung này tập trung vào **Các loại game server** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Giải thích được **Các loại game server** trong bối cảnh server-side game development.
- Biết chủ đề này nằm ở đâu trong game backend architecture.
- Tạo được ghi chú hoặc demo nhỏ để đưa vào portfolio.

## 3. Nội dung roadmap
- Turn-based server
- Real-time authoritative server
- Lobby/matchmaking server
- Chat server
- World server / zone server
- Backend service cho game mobile

## 4. Bài tập thực hành
- Tóm tắt bài học trong 5 dòng bằng lời của bạn.
- Tạo một demo hoặc architecture note nhỏ.
- Liên hệ bài học với một capstone project cuối khóa.

## 5. Artifact nên tạo
- Study note
- Prototype
- Portfolio README

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Các loại game server** trong kiến trúc game backend không?
- Chủ đề này ảnh hưởng đến latency, reliability, scaling, consistency, security hay observability?
- Artifact nào từ bài này có thể đưa vào portfolio server-side game developer?

## 7. Tổng kết
**Các loại game server** nên được học bằng cách thiết kế, code thử, đo đạc và ghi lại trade-off. Hãy biến bài học thành một demo, diagram hoặc README có thể review.
