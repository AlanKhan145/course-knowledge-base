## 1. Tóm tắt
Nội dung này tập trung vào **Kỹ năng cần có** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Giải thích được **Kỹ năng cần có** trong bối cảnh server-side game development.
- Biết chủ đề này nằm ở đâu trong game backend architecture.
- Tạo được ghi chú hoặc demo nhỏ để đưa vào portfolio.

## 3. Nội dung roadmap
- Network programming
- Concurrency/multithreading
- Database/cache
- Cloud/container
- Message queue
- Observability/debugging
- Security

## 4. Bài tập thực hành
- Tóm tắt bài học trong 5 dòng bằng lời của bạn.
- Tạo một demo hoặc architecture note nhỏ.
- Liên hệ bài học với một capstone project cuối khóa.

## 5. Artifact nên tạo
- Study note
- Prototype
- Portfolio README

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Kỹ năng cần có** trong kiến trúc game backend không?
