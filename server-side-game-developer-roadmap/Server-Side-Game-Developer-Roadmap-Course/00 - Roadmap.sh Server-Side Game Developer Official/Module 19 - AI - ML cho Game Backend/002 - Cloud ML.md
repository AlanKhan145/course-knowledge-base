# 002 - Cloud ML

**Module:** Module 19 - AI/ML cho Game Backend
**Roadmap item:** Bài 73

**Thứ tự trong module:** 002
**Thời lượng gợi ý:** 40-60 phút

## 1. Tóm tắt
Nội dung này tập trung vào **Cloud ML** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Áp dụng **Cloud ML** để vận hành game backend có region, scaling, health check và monitoring.
- Biết đóng gói service, cấu hình môi trường và triển khai an toàn.
- Tạo deployment checklist hoặc Docker/Kubernetes note cho project.

## 3. Nội dung roadmap
- Amazon ML
- Azure ML
- ML pipeline
- Model serving
- Online/offline inference

## 4. Bài tập thực hành
- Tạo Docker/Docker Compose hoặc deployment diagram cho service nhỏ.
- Thêm health check, graceful shutdown và config/secrets note.
- Ghi lại chiến lược scale theo region, room hoặc worker.

## 5. Artifact nên tạo
- Docker/Kubernetes note
- Deployment checklist
- Monitoring plan

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Cloud ML** trong kiến trúc game backend không?
- Chủ đề này ảnh hưởng đến latency, reliability, scaling, consistency, security hay observability?
- Artifact nào từ bài này có thể đưa vào portfolio server-side game developer?

## 7. Tổng kết
**Cloud ML** nên được học bằng cách thiết kế, code thử, đo đạc và ghi lại trade-off. Hãy biến bài học thành một demo, diagram hoặc README có thể review.
