## 1. Tóm tắt
Nội dung này tập trung vào **ORM & DAL** trong lộ trình Server-Side Game Developer. Sau phần này, bạn nên hiểu vai trò của chủ đề trong game backend và tạo được một artifact kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Dùng **ORM & DAL** để lưu account, inventory, profile, leaderboard hoặc telemetry đúng nhu cầu.
- Biết trade-off consistency, latency, indexing, cache và migration.
- Tạo schema/cache/query note cho một service game backend.

## 3. Nội dung roadmap
- ORM là gì?
- Data Access Layer
- Repository pattern
- Migration
- Query optimization

## 4. Bài tập thực hành
- Thiết kế bảng/collection/cache key cho profile, inventory hoặc leaderboard.
- Ghi query chính, index cần có và policy cache invalidation.
- Nêu cách backup, migration hoặc consistency khi có giao dịch vật phẩm.

## 5. Artifact nên tạo

- Query/index note
- Data consistency checklist
