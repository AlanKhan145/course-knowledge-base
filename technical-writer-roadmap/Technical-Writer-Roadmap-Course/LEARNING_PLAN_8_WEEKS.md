# Lộ trình học đề xuất 8 tuần

Kế hoạch này chia roadmap Technical Writer thành 8 tuần học và một sản phẩm đầu ra mỗi tuần.

| Tuần | Chủ đề chính                          | Sản phẩm đầu ra                                 |
| ---- | ------------------------------------- | ----------------------------------------------- |
| 1    | Nhập môn Technical Writing            | Bài viết: “Technical Writer là gì?”             |
| 2    | Kỹ năng viết & style guide            | Viết lại một bài docs khó hiểu thành bản dễ đọc |
| 3    | Markdown, Git, Docs-as-code           | Tạo repo docs bằng Markdown                     |
| 4    | Product docs & How-to guides          | Viết 1 how-to guide hoàn chỉnh                  |
| 5    | Developer docs & API reference        | Viết tài liệu cho 1 REST API mẫu                |
| 6    | Help docs & troubleshooting           | Viết troubleshooting guide cho 5 lỗi thường gặp |
| 7    | Technical content marketing & SEO     | Viết 1 technical blog post có keyword + CTA     |
| 8    | Analytics, optimization, distribution | Tạo portfolio technical writing gồm 5 mẫu bài   |
