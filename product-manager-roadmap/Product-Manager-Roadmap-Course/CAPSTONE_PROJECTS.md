# Capstone Projects

Các project cuối khóa giúp gom discovery, strategy, planning, launch, metrics, stakeholder và risk management thành portfolio Product Manager.

## Project 01 - B2C Onboarding Activation Case

File chi tiết: [B2C Onboarding Activation Case](99 - Portfolio Capstone Projects/Project 01 - B2C Onboarding Activation Case.md)

Một ứng dụng consumer có nhiều signup nhưng activation thấp. Bạn cần tìm vấn đề, đề xuất MVP cải thiện onboarding và đo tác động.

Deliverable chính:
- Problem framing
- User interview plan
- Activation metrics tree
- PRD cho onboarding experiment
- A/B test plan

## Project 02 - B2B SaaS Analytics Dashboard

File chi tiết: [B2B SaaS Analytics Dashboard](99 - Portfolio Capstone Projects/Project 02 - B2B SaaS Analytics Dashboard.md)

Một sản phẩm B2B cần dashboard giúp khách hàng thấy value rõ hơn và giảm churn.

Deliverable chính:
- Persona and jobs to be done
- Competitive analysis
- Dashboard requirements
- Roadmap now/next/later
- Launch and enablement plan

## Project 03 - Marketplace Discovery and MVP

File chi tiết: [Marketplace Discovery and MVP](99 - Portfolio Capstone Projects/Project 03 - Marketplace Discovery and MVP.md)

Bạn đang đánh giá một ý tưởng marketplace hai phía và cần quyết định có nên build MVP hay không.

Deliverable chính:
- Market segmentation
- Supply/demand assumptions
- Smoke test
- Concierge MVP
- Risk register

## Project 04 - AI-Assisted Productivity Feature

File chi tiết: [AI-Assisted Productivity Feature](99 - Portfolio Capstone Projects/Project 04 - AI-Assisted Productivity Feature.md)
