## Thực hành trọng tâm
Tạo wireframe/prototype thô, chạy usability test với 2-3 người và ghi lại friction chính.

## Output sau module
- Wireframe, prototype test script and usability findings note
- Ghi chú quyết định hoặc insight có thể chia sẻ với stakeholder
- Danh sách câu hỏi còn mở cho research, strategy hoặc execution
