# Module 13 - Keep Learning — Học tiếp

## Mục tiêu module
Biết định hướng học tiếp sau khóa Product Manager Roadmap.

## Bài học

## Thực hành trọng tâm
Chọn roadmap học tiếp, đặt mục tiêu 30 ngày và artifact cần hoàn thành.

## Output sau module
- Personal PM learning roadmap
- Ghi chú quyết định hoặc insight có thể chia sẻ với stakeholder
- Danh sách câu hỏi còn mở cho research, strategy hoặc execution
