## Thực hành trọng tâm
Biến một problem đã validate thành PRD nhỏ, roadmap now/next/later và 5 user stories.

## Output sau module
- PRD, roadmap slice, backlog and user story map
- Ghi chú quyết định hoặc insight có thể chia sẻ với stakeholder
- Danh sách câu hỏi còn mở cho research, strategy hoặc execution
