# Setup and A Challenge For You

## Overview
This lecture walks through the initial project setup for the Shopping List App and presents the first challenge for you to attempt independently. You will configure the Flutter project structure, create necessary files, and get a clear understanding of what needs to be built before watching the solution. The challenge is designed to reinforce your Flutter fundamentals before new concepts are introduced.

## Key Points
- Setting up a new Flutter project with a clean folder structure is an important first step
- The challenge involves defining a data model and building the basic UI for a shopping list
- Attempting challenges before watching solutions significantly improves retention
- The project uses a standard `lib/` folder structure with separate files for models, screens, and widgets
- Understanding the expected final output of the challenge helps guide your implementation

## Tips
- Sketch the UI on paper before writing code — think about what widgets you need
- Break the challenge into small steps: model first, then UI, then interactivity
- If you get stuck, re-read the challenge requirements carefully before looking at the solution

## Notes
The initial setup phase is often underestimated but a well-organized project structure makes future development much smoother. This challenge tests your understanding of Dart classes, Flutter widgets, and basic list rendering. Treat it as a checkpoint to identify any gaps in your knowledge before proceeding.

## Summary
The setup lecture provides project scaffolding and challenges you to independently build the Shopping List App's foundation before seeing the guided solution.
