# 002 - What Is Flutter

## Section

Introduction

## Main Idea

Flutter is a cross-platform UI framework and toolset created by Google. It allows developers to build applications for multiple platforms from one single codebase.

The main goal of Flutter is to make it possible to write app code once and then generate apps for different target platforms such as Android and iOS. Instead of learning different programming languages and tools for every platform, developers can use Flutter and Dart to build multi-platform apps more efficiently.

## What Is Flutter?

Flutter is made of two main parts:

1. **A UI framework**
2. **A set of development tools**

Together, these two parts allow developers to create beautiful user interfaces and then convert their code into apps that can run on different platforms.

## Part 1: Flutter As A UI Framework

Flutter is a UI framework, which means it provides ready-to-use code packages and building blocks that developers can use to create user interfaces.

These building blocks help developers create:

* Text
* Buttons
* Layouts
* Screens
* Forms
* Navigation
* Visual components
* Complete app interfaces

In Flutter, user interfaces are built using widgets. A widget is a reusable UI building block. Almost everything visible on the screen in a Flutter app is a widget.

## Part 2: Flutter As A Toolset

Flutter is not only a collection of UI components. It also includes tools that convert the code written by developers into code that can run on real devices.

The Flutter code written by the developer does not automatically run on every platform by itself. It must be translated into machine code that each target platform understands.

For example:

* Android needs code that can run on Android devices.
* iOS needs code that can run on iPhones and iPads.

Flutter provides the tools needed to perform this translation process.

## Cross-Platform Development

One of Flutter’s biggest advantages is cross-platform development.

Without a solution like Flutter, developers often need to learn different languages and tools for different platforms. For example, native Android development and native iOS development traditionally require different technologies.

Flutter solves this problem by allowing developers to use:

* One codebase
* One programming language
* One UI framework
* Multiple target platforms

This makes development faster and more efficient.

## Why Flutter Is Useful

Flutter is useful because it helps developers build apps for multiple platforms without rewriting the same app several times.

With Flutter, developers can:

* Build Android and iOS apps from one codebase.
* Create consistent user interfaces across platforms.
* Use one programming language instead of multiple languages.
* Develop apps more efficiently.
* Maintain one shared project instead of separate platform-specific projects.

## Key Concepts

### UI Framework

A UI framework provides tools, components, and patterns for building user interfaces.

### Single Codebase

A single codebase means the developer writes one main set of code that can be used to generate apps for different platforms.

### Target Platform

A target platform is the system where the app will run, such as Android, iOS, web, desktop, or other supported platforms.

### Machine Code

Machine code is platform-specific code that can be executed by a device. Flutter tools help convert the developer’s Flutter code into code that works on each target platform.

### Widgets

Widgets are the core building blocks of Flutter user interfaces. Text, buttons, layouts, and screens are all represented as widgets.

## Important Notes

* Flutter is more than just a UI framework.
* Flutter includes both code packages and development tools.
* The UI framework helps developers build interfaces.
* The Flutter tools help convert the code into platform-specific apps.
* Flutter is designed for building multi-platform applications.
* Developers only need to learn one main programming language for Flutter app development: Dart.

## Mental Model

A simple way to understand Flutter is:

```txt
Flutter code + Flutter UI framework
        ↓
Flutter tools
        ↓
Platform-specific app
        ↓
Android / iOS / other platforms
```

Flutter gives developers both the building blocks for creating the app and the tools for turning that app into something that can run on real devices.

## Why This Lecture Matters

This lecture explains the basic identity of Flutter. Before learning how to write Flutter code, students need to understand what Flutter actually is and what problem it solves.

The key idea is that Flutter helps developers avoid writing separate apps for every platform. Instead, they can focus on one shared codebase and use Flutter to generate apps for multiple platforms.

## Summary

Flutter is a UI framework and development toolset for building cross-platform applications from one codebase. It provides reusable UI building blocks for creating app interfaces and tools that convert the written code into platform-specific apps.

The main benefit of Flutter is that developers can build apps for multiple platforms while learning and maintaining only one primary codebase and programming language.
