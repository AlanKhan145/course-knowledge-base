# 001 - Welcome To This Course

## Section

Introduction

## Main Idea

This lecture introduces the Flutter & Dart course and explains what students can expect to learn. The instructor, Maximilian Schwarzmüller, welcomes learners and presents the course as a beginner-friendly path to building mobile apps for Android and iOS using Flutter and Dart.

The lecture also highlights that students do not need prior Flutter or Dart experience. Even programming knowledge is not strictly required, although it can be helpful. The course starts from the basics and gradually moves toward more advanced Flutter concepts.

## Learning Objectives

By the end of this lecture, students should understand:

* Who the instructor is.
* What the course is about.
* Why Flutter and Dart are useful for mobile app development.
* That the course is suitable for beginners.
* That the learning path will be project-based.
* That students will build multiple demo apps throughout the course.

## Key Concepts

### Flutter

Flutter is the main framework taught in this course. It allows developers to build mobile applications for Android and iOS from a single codebase.

### Dart

Dart is the programming language used with Flutter. Students will learn Dart as part of the course, starting from the basics.

### Cross-Platform Development

One of Flutter’s biggest advantages is that it lets developers create apps for multiple platforms without writing separate native code for each platform.

### Project-Based Learning

The course does not only teach theory. Students will learn Flutter and Dart by building real demo applications, which helps them apply concepts in practical scenarios.

## Important Notes

* No prior Flutter knowledge is required.
* No prior Dart knowledge is required.
* Programming experience can help, but it is not mandatory.
* The course starts from the ground up.
* Students will gradually move from basic concepts to advanced Flutter features.
* The demo apps built throughout the course can become part of a beginner portfolio.

## Instructor’s Message

The instructor emphasizes that the course is designed to help students learn step by step. By the end of the course, students should be able to build their own Flutter apps independently.

The course is practical and hands-on, meaning students will not only watch explanations but also build real apps while learning.

## Why This Lecture Matters

This welcome lecture gives students the right mindset before starting the course. It explains the course structure, sets expectations, and motivates learners to continue.

It also prepares students for the first major question of the course:

**What exactly is Flutter?**

## Tips For Students

* Watch the introductory lectures before jumping into coding.
* Set up a regular study schedule.
* Take notes while watching the lectures.
* Build the demo apps instead of only watching the videos.
* Use the Q&A or community support when stuck.
* Treat each project as part of your learning portfolio.

## Summary

This lecture welcomes students to the Flutter & Dart course and introduces the overall learning journey. Students will learn how to build Android and iOS apps using Flutter and Dart, starting from the basics and moving toward advanced topics through practical demo projects.

The course is beginner-friendly, project-based, and designed to help students become confident Flutter developers.
