# A Challenge For You

## Overview

This lecture introduces a hands-on coding challenge for the Quiz App project. The goal is to apply the Flutter and Dart fundamentals learned in the previous module by creating the app’s starting screen from scratch.

Students are encouraged to complete the challenge independently before watching the next lecture, where the instructor builds the same screen step by step.

## Key Points

* A new Flutter project should be created for this module.
* The `main.dart` file should be cleared so the app can be built from scratch.
* A zipped assets folder is provided and should be added to the project.
* The assets folder contains an image that will be used later in the module.
* The challenge is to build the starting screen of the Quiz App independently.
* The goal is to practice creating widgets, layouts, and basic UI structure.
* The next lecture will provide the instructor’s solution for comparison.
* The button on the starting screen is not a `TextButton`; another Flutter button widget should be used instead.

## Setup Instructions

Before starting the challenge:

1. Create a new Flutter project using the `flutter create` command.
2. Choose a project name, such as `advanced_basics`, or any other suitable name.
3. Open the project.
4. Go to the `lib` folder.
5. Open the `main.dart` file.
6. Delete all existing content from `main.dart`.
7. Unzip the provided assets folder.
8. Add the assets folder to your Flutter project.

## Challenge Task

Your challenge is to build the starting screen of the Quiz App on your own.

You should write all the code needed to:

* Start the Flutter app.
* Display a user interface on the screen.
* Build the starting screen layout.
* Apply basic styling.
* Add a button that matches the design shown in the course.
* Use the Flutter knowledge gained from the previous module.

## Hint

The button shown on the starting screen is not the same `TextButton` used in the previous module.

To find the right button widget, you can check the Flutter Widget Catalog, especially the Material Components section. This will help you explore the different button widgets available in Flutter and choose the one that best matches the design.

## Recommended Approach

* Pause the video before watching the solution.
* Try to complete the challenge independently.
* Sketch the screen layout before writing code.
* Use the Flutter documentation when needed.
* Do not worry if your solution is different from the instructor’s.
* Compare your solution with the next lecture to understand different possible approaches.

## Tips

* Focus on practicing the basics rather than writing perfect code.
* Start with the app structure first, then improve the layout.
* Use widgets you already know from the previous module.
* Look up unfamiliar widgets instead of immediately watching the solution.
* Even a partial attempt is more valuable than skipping the challenge.

## Notes

This challenge is designed to reinforce the fundamentals from the previous course section. Building the screen independently helps strengthen your understanding of how Flutter apps are structured and how widgets are combined to create a user interface.

The next lecture will walk through the solution from scratch, allowing you to compare your implementation with the instructor’s version.

## Summary

This lecture gives you a practical challenge: create the starting screen of the Quiz App from scratch in a new Flutter project. The purpose is to practice Flutter basics, widget creation, layout composition, and UI styling before moving on to the instructor’s solution in the next lecture.
