# Module Introduction

## Overview
This module introduces Firebase integration with Flutter, covering authentication, image storage, Firestore database, and push notifications. You will build a fully functional chat application that uses Firebase services as the backend. By the end, you will understand how to connect a Flutter app to Firebase and leverage its real-time capabilities.

## Key Points
- Firebase is a Backend-as-a-Service (BaaS) platform by Google that simplifies backend development
- This module covers Firebase Authentication, Cloud Firestore, Firebase Storage, and Cloud Messaging
- The project is a chat app where users can sign up, log in, upload a profile image, and send messages
- No custom backend server is required — Firebase handles all server-side logic

## Tips
- Review the Firebase documentation at firebase.google.com before starting
- Ensure you have a Google account to create a Firebase project
- The FlutterFire CLI will be used to configure Firebase — install it before proceeding

## Notes
Firebase's free Spark plan is sufficient for development and learning purposes. The module progresses from basic auth setup to advanced features like push notifications and cloud functions. Keep the Firebase console open alongside your IDE throughout this module.

## Summary
This module walks through building a Firebase-powered Flutter chat app covering auth, storage, Firestore, and push notifications end to end.
