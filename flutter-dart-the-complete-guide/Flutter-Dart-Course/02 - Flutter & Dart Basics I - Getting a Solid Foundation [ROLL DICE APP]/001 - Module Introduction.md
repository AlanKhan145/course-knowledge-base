# Module Introduction

## Overview

This lecture introduces the second major module of the Flutter & Dart course. In this section, students will build their first basic Flutter app: a Roll Dice app. The app allows users to roll a dice and generate random values, which are then displayed through the correct dice image on the screen.

Although the app is simple, it is an excellent starting point for learning Flutter and Dart because it introduces many core concepts that will be used throughout the rest of the course.

## Learning Goals

By the end of this module, students will understand the essential foundations of Flutter and Dart development, including how to:

* Build a basic Flutter app step by step
* Combine widgets to create a user interface
* Use Dart syntax in a real app
* Work with variables, functions, and classes
* Update the UI based on user interaction
* Understand the role of state in Flutter apps
* Use stateful widgets to create dynamic behavior

## Key Points

* The practical project in this module is a Roll Dice app.
* The app generates a random dice value and shows the matching dice image.
* This module focuses on Flutter and Dart fundamentals.
* Flutter apps are built by combining widgets.
* Understanding widgets is one of the most important foundations for Flutter development.
* Even a simple app can teach many important concepts, such as layout, logic, interactivity, and state.

## Why This Module Matters

The Roll Dice app may look basic, but it introduces concepts that are essential for almost every Flutter app. Instead of only learning theory, students will apply Flutter and Dart features directly in a real project.

This module helps build a strong foundation before moving on to more advanced topics in later sections of the course.

## Tips

* Code along with the instructor instead of only watching the videos.
* Take notes on important Flutter and Dart concepts.
* Focus on understanding widgets deeply, because Flutter development is mostly about combining widgets.
* Do not rush through this section; the fundamentals here will be reused many times later.
* Experiment with the Roll Dice app after completing it, such as changing the design, dice images, or layout.

## Summary

This module introduces Flutter and Dart fundamentals through a hands-on Roll Dice app. Students will learn how Flutter apps are built by combining widgets, writing Dart code, handling interactivity, and updating the UI based on changing data.
