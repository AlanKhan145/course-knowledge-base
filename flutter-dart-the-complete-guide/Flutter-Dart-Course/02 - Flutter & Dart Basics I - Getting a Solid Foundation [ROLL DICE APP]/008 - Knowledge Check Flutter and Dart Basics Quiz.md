# Knowledge Check Flutter and Dart Basics Quiz

## Overview
This quiz section tests comprehension of the fundamental Flutter and Dart concepts covered so far in the module. It reinforces understanding of functions, imports, app startup, compilation, and project structure. Completing the quiz helps identify any gaps in knowledge before advancing to more complex widget topics.

## Key Points
- Quiz covers Dart functions, return types, and the `main()` entry point
- Questions address how Flutter apps start with `runApp()` and the widget tree
- Import syntax for Dart standard library vs Flutter packages is tested
- JIT vs AOT compilation concepts may appear in quiz questions
- Project structure knowledge (lib folder, pubspec.yaml) is assessed

## Tips
- Review the role of `main()` and `runApp()` before attempting the quiz
- Remember the difference between `void` functions (no return) and typed functions (return a value)
- Recall that `pubspec.yaml` manages dependencies and `lib/main.dart` is the app entry point
- JIT = development/hot reload; AOT = production/optimized native code
- If unsure, re-watch the relevant lecture rather than guessing
