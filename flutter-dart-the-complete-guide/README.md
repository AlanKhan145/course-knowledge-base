# Flutter & Dart: The Complete Guide

Comprehensive notes from a hands-on Flutter course. Each section builds a real app to reinforce concepts.

## Course Structure

### Section 1 — Introduction
- Flutter overview, setup on Windows & macOS

### Section 2 — Flutter & Dart Basics I `[Roll Dice App]`
- Dart fundamentals: variables, functions, classes
- Core widgets, MaterialApp, Scaffold

### Section 3 — Flutter & Dart Basics II `[Quiz App]`
- Stateless vs Stateful widgets
- State management basics, lists and maps

### Section 4 — Debugging Flutter Apps
- Flutter DevTools, breakpoints, common errors

### Section 5 — Interactivity, Widgets & Theming `[Expense Tracker App]`
- Custom widgets, theming, date pickers, charts

### Section 6 — Responsive & Adaptive UI `[Expense Tracker App]`
- `LayoutBuilder`, `MediaQuery`, platform-specific UI

### Section 7 — Flutter & Dart Internals `[Todo App]`
- Widget tree, element tree, render tree
- `const` widgets, performance optimization

### Section 8 — Multi-Screen Navigation `[Meals App]`
- Named routes, `Navigator`, passing data between screens

### Section 9 — State Management with Riverpod
- Providers, `ConsumerWidget`, `StateNotifierProvider`

### Section 10 — Animations
- Implicit animations (`AnimatedContainer`, `AnimatedOpacity`)
- Explicit animations (`AnimationController`, `Tween`)

### Section 11 — Forms & User Input `[Shopping List App]`
- `Form`, `TextFormField`, validation, global keys

### Section 12 — HTTP & Networking
- `http` package, REST APIs, `FutureBuilder`

### Section 13 — Native Device Features `[Place Finder App]`
- Camera & image picker, Google Maps, SQLite local storage

### Section 14 — Firebase, Auth & Push Notifications `[Chat App]`
- Firebase Auth, Firestore, Cloud Storage, FCM

### Section 15 — Roundup & Bonus
- Course recap, next steps
