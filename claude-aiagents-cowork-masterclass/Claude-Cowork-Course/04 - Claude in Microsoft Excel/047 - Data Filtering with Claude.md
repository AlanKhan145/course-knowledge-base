# 047 - Data Filtering with Claude

## Section

Claude in Microsoft Excel

## Main Idea

Filter data by conditions using Claude in Excel to isolate specific subsets.

## Learning Objectives

By the end of this lecture, students should understand:

* Define filtering conditions in plain language.
* Request filters on multiple columns simultaneously.
* Ask Claude to count or summarize the filtered results.
* Save filtered subsets for reporting.
