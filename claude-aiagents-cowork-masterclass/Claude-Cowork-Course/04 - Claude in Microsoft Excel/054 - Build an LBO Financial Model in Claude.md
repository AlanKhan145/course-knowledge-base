## Section

## Main Idea

Use Claude to generate or assist with building a leveraged buyout model in Excel.

## Learning Objectives

By the end of this lecture, students should understand:

* Provide entry assumptions: purchase price, debt structure, EBITDA.
* Ask Claude to model debt amortization and interest expense.
* Request operating performance projections.
* Ask for exit scenario analysis and IRR/MOIC calculation.
