## Section

## Main Idea

Set up the Gemini API key and configuration needed for the AI-powered calorie tracker application.

## Learning Objectives

By the end of this lecture, students should understand:

* Obtain a Gemini API key from Google AI Studio.
* Store the key securely in an environment variable.
* Configure the project to access the key without hardcoding.
* Verify API connectivity before proceeding to app building.
