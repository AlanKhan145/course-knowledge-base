# 090 - CLAUDE.md Best Practices

## Section

Claude Code for Building Apps

## Main Idea

Use CLAUDE.md to give Claude Code stable project instructions that persist across sessions.

## Learning Objectives

By the end of this lecture, students should understand:

* CLAUDE.md tells Claude Code what the project is, what rules to follow, and what not to do.
* Good CLAUDE.md files include project goal, tech stack, folder structure, coding rules, test commands, style constraints, and known gotchas.
* Keep CLAUDE.md up to date as the project evolves.
* CLAUDE.md is read at the start of every Claude Code session.
