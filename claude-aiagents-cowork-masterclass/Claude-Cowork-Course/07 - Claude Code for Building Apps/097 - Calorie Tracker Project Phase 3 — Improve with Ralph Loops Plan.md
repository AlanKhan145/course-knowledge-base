# 097 - Calorie Tracker Project Phase 3 — Improve with Ralph Loops Plan

## Section

Claude Code for Building Apps

## Main Idea

Plan the iterative improvement loops that will refine the calorie tracker application.

## Learning Objectives

By the end of this lecture, students should understand:

* Identify the top three areas for improvement: UX, accuracy, performance.
* Define what 'good enough' means for each area.
* Plan the improvement loop: test, identify gap, prompt Claude, review, repeat.
* Set a time box for the improvement phase.
