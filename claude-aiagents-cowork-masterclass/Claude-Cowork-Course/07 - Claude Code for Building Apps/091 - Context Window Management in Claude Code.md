# 091 - Context Window Management in Claude Code

## Section

Claude Code for Building Apps

## Main Idea

Manage context effectively for long coding tasks to prevent quality degradation.

## Learning Objectives

By the end of this lecture, students should understand:

* Start new sessions at meaningful milestones to keep context fresh.
* Summarize key decisions in CLAUDE.md before ending a session.
* Keep requirements in files rather than in the chat history.
* Use smaller, focused tasks rather than large open-ended sessions.
