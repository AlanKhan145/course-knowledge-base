# 095 - Calorie Tracker Project Phase 2 — Build API Routes Plan

## Section

Claude Code for Building Apps

## Main Idea

Plan the backend API routes needed to support all calorie tracker features.

## Learning Objectives

By the end of this lecture, students should understand:

* Identify all operations the app needs: text entry, image upload, quick add, history.
* Design RESTful API routes for each operation.
* Plan request and response schemas.
* Document the API plan in CLAUDE.md before building.
