## Main Idea

## Learning Objectives

By the end of this lecture, students should understand:

* MCP provides a structured interface between model and tools.
* Connectors allow Claude to interact with Gmail, Calendar, Drive, Slack, or Salesforce.
* Tool access should be permissioned and reviewed before use.
