## Main Idea

## Learning Objectives

By the end of this lecture, students should understand:

* Skills encode purpose, instructions, constraints, and tool usage guidance.
* A skill can be reused across multiple projects.
* Skills reduce the need to re-explain workflows to Claude each session.
