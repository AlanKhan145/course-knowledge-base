# 015 - Claude Plugins

## Section

Claude Cowork, Skills & Plugins Mastery for Workflow Automation

## Main Idea

Understand plugins as larger packages that bundle skills, connectors, slash commands, and subagents into a cohesive workflow system.

## Learning Objectives

By the end of this lecture, students should understand:

* A skill is one reusable capability; a plugin is a bundle of capabilities.
* Plugins can coordinate multiple tools and agents.
* Plugins are designed for specific domains like finance, data, or marketing.
