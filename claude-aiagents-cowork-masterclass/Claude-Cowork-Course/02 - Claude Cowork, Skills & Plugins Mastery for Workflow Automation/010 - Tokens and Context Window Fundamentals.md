## Section

Claude Cowork, Skills & Plugins Mastery for Workflow Automation

## Main Idea

Understand tokens and context windows as the foundational resource model for all Claude interactions.

## Learning Objectives

By the end of this lecture, students should understand:

* A token is a chunk of text processed by the model.
* The context window is the maximum input/output the model can consider at once.
* Good context management is essential for long projects.
