## Main Idea

Claude Cowork is introduced as an AI coworking environment that can work with local folders and perform multi-step workflows.

## Learning Objectives

By the end of this lecture, students should understand:

* Cowork is not only chat; it is a task-execution environment.
* Claude can work on files in your local system.
* Multi-step workflows are coordinated automatically.
