# 139 - Build Morning Brief Automation

## Section

Automation Blueprints Part 1

## Main Idea

Create a daily morning summary that consolidates key information to start the day.

## Learning Objectives

By the end of this lecture, students should understand:

* Define the morning brief inputs: calendar, tasks, news, weather, priorities.
* Write the automation instructions for brief generation.
* Format the output as a scannable daily digest.
* Test the brief with current real data.
* Schedule or trigger the brief each morning.
