# 155 - Build Content Machine Automation

## Section

Automation Blueprints Part 2

## Main Idea

Build a content generation and repurposing workflow that maximizes content output from minimal input.

## Learning Objectives

By the end of this lecture, students should understand:

* Define content inputs: core ideas, raw notes, existing pieces.
* Write automation instructions for multi-format content generation.
* Add repurposing logic: blog to LinkedIn to tweet thread.
* Connect to brand folder for voice and style consistency.
* Test with a content idea from your vault.
