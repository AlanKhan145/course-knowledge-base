# 157 - Build Weekly Executive Reporting Automation

## Section

Automation Blueprints Part 2

## Main Idea

Build an executive report workflow that synthesizes a week's work into a concise, shareable summary.

## Learning Objectives

By the end of this lecture, students should understand:

* Define report inputs: completed tasks, metrics, decisions, blockers, next week plan.
* Write automation instructions for executive summary generation.
* Format the report for executive readers.
* Connect to relevant data sources: project tools, metrics.
* Test with a real week of work.
