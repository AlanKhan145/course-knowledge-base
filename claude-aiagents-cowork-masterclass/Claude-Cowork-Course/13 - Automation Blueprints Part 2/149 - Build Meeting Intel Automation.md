# 149 - Build Meeting Intel Automation

## Section

Automation Blueprints Part 2

## Main Idea

Prepare comprehensive meeting intelligence briefs automatically before each meeting.

## Learning Objectives

By the end of this lecture, students should understand:

* Define meeting intel inputs: attendee list, meeting purpose, context documents.
* Write automation instructions for brief generation.
* Connect to calendar for meeting data.
* Test with an upcoming real meeting.
* Assess brief quality and completeness.
