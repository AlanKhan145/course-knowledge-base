# 151 - Build Email Triage Automation

## Section

Automation Blueprints Part 2

## Main Idea

Create an email prioritization and response-drafting workflow to manage inbox overload.

## Learning Objectives

By the end of this lecture, students should understand:

* Define triage criteria: urgency, sender importance, required action.
* Write automation instructions for email categorization.
* Add response-drafting capability for priority emails.
* Connect to Gmail via MCP.
* Test with a sample inbox snapshot.
