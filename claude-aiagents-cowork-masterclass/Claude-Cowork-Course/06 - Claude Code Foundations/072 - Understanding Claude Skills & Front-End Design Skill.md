# 072 - Understanding Claude Skills & Front-End Design Skill

## Section

Claude Code Foundations

## Main Idea

Learn how skill.md files guide Claude Code output quality and consistency for frontend development.

## Learning Objectives

By the end of this lecture, students should understand:

* A skill.md file encodes design rules, component patterns, and code conventions.
* The frontend design skill includes layout rules, typography, color usage, and component structure.
* Skills can be added to any project folder.
* Skill quality directly affects output quality.
