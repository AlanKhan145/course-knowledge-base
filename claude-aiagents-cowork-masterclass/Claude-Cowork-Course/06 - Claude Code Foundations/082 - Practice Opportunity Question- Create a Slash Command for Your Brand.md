# 082 - Practice Opportunity Question: Create a Slash Command for Your Brand

## Section

Claude Code Foundations

## Main Idea

Create a brand-specific slash command that automates a content or design workflow for your brand.

## Learning Objectives

By the end of this lecture, students should understand:

* Choose a brand-specific repetitive task.
* Define the command with brand tone, colors, fonts, and vocabulary.
* Test the command with real brand content.
* Document the command for team sharing.
