# 075 - Why Agents and Claude Folders Contain the Same skill.md

## Section

Claude Code Foundations

## Main Idea

Understand why skills can exist in both agent folders and Claude folders and how this enables consistent behavior.

## Learning Objectives

By the end of this lecture, students should understand:

* Skills in Claude folders guide all work in that folder.
* Skills in agent folders guide that specific agent's behavior.
* The same skill file can govern both contexts.
* This enables consistent behavior across Cowork and Code workflows.
