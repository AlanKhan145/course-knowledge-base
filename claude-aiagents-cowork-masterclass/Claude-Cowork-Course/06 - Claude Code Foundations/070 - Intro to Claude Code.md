# 070 - Intro to Claude Code

## Section

Claude Code Foundations

## Main Idea

Understand Claude Code as an agentic coding environment that reads, writes, and executes code inside real projects.

## Learning Objectives

By the end of this lecture, students should understand:

* Claude Code is more than a code generator: it reads files, edits code, and runs commands.
* It works inside your actual project directory.
* It maintains context across multi-step coding sessions.
* It supports human review before executing risky changes.
