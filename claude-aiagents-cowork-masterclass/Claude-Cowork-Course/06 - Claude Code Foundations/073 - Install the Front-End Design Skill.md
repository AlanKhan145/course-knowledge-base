# 073 - Install the Front-End Design Skill

## Section

Claude Code Foundations

## Main Idea

Install and verify the frontend design skill in a Claude Code project.

## Learning Objectives

By the end of this lecture, students should understand:

* Download the frontend design skill file.
* Place skill.md in the correct project folder location.
* Verify Claude Code recognizes the skill.
* Test the skill with a simple component generation request.
