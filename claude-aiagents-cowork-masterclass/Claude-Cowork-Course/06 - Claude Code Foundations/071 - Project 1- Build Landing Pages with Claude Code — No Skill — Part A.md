## Section

## Main Idea

Build a landing page using only raw Claude Code instructions, without any frontend design skill.

## Learning Objectives

By the end of this lecture, students should understand:

* Create a project folder and CLAUDE.md file.
* Describe the landing page requirements to Claude Code.
* Review and test the generated HTML and CSS.
* Iterate on the design with follow-up prompts.
