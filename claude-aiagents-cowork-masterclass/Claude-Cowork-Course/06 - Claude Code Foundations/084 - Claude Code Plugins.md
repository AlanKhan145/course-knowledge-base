# 084 - Claude Code Plugins

## Section

Claude Code Foundations

## Main Idea

Learn how plugins extend Claude Code workflows by bundling skills, commands, and connectors.

## Learning Objectives

By the end of this lecture, students should understand:

* Claude Code plugins work similarly to Cowork plugins.
* Plugins can automate multi-step research, assessment, or development workflows.
* Browser automation tools like Playwright can be integrated via plugins.
* Plugins encapsulate complex workflows behind simple triggers.
