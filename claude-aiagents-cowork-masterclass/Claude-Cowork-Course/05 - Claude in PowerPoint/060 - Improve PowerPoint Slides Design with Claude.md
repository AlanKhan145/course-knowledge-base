# 060 - Improve PowerPoint Slides Design with Claude

## Section

Claude in PowerPoint

## Main Idea

Use Claude to improve layout, visual hierarchy, and design consistency across a slide deck.

## Learning Objectives

By the end of this lecture, students should understand:

* Ask Claude to audit the current design for consistency issues.
* Request specific improvements: alignment, whitespace, font hierarchy.
* Apply Claude's design suggestions selectively.
* Compare before and after versions for quality assessment.
