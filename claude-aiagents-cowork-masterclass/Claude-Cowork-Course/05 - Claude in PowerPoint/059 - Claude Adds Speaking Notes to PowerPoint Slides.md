# 059 - Claude Adds Speaking Notes to PowerPoint Slides

## Section

Claude in PowerPoint

## Main Idea

Add detailed presenter notes to existing or newly generated slides.

## Learning Objectives

By the end of this lecture, students should understand:

* Upload or open the slide deck in Claude.
* Ask Claude to generate speaker notes for each slide.
* Specify the presentation context and audience.
* Review notes for accuracy and adjust tone as needed.
