# 063 - Summarize and Organize Existing PowerPoint Slides

## Section

Claude in PowerPoint

## Main Idea

Improve existing decks by summarizing and restructuring slides for better narrative flow.

## Learning Objectives

By the end of this lecture, students should understand:

* Upload the existing deck to Claude.
* Ask Claude to identify redundant or unclear slides.
* Request a restructured narrative order.
* Ask for summary slides to replace overly detailed sections.
