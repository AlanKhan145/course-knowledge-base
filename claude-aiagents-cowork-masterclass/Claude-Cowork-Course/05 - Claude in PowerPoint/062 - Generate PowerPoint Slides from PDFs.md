# 062 - Generate PowerPoint Slides from PDFs

## Section

Claude in PowerPoint

## Main Idea

Turn PDFs into slide decks by extracting key information and structuring it as slides.

## Learning Objectives

By the end of this lecture, students should understand:

* Upload a PDF to Claude.
* Ask Claude to identify the key points suitable for slides.
* Request a structured slide outline from the PDF content.
* Generate slides and refine the design and content.
