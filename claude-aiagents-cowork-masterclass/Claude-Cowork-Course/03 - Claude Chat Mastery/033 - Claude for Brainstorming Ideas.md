# 033 - Claude for Brainstorming Ideas

## Section

Claude Chat Mastery

## Main Idea

Generate, filter, and refine ideas using Claude as a structured brainstorming partner.

## Learning Objectives

By the end of this lecture, students should understand:

* Ask for quantity first, then filter for quality.
* Request ideas from multiple angles or stakeholder perspectives.
* Use follow-up prompts to develop the strongest ideas.
* Ask Claude to steelman or challenge each idea.
