# 029 - Prompt Engineering Fundamentals Explained

## Section

Claude Chat Mastery

## Main Idea

Learn the foundations of clear, effective prompting that produce reliable and high-quality Claude outputs.

## Learning Objectives

By the end of this lecture, students should understand:

* Clear prompts specify role, task, context, constraints, output format, examples, and evaluation criteria.
* Vague prompts produce vague outputs.
* Structured prompts are reusable and improvable over time.
