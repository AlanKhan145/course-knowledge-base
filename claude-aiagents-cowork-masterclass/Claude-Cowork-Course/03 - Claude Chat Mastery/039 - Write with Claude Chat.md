## Section

Claude Chat Mastery

## Main Idea

Use Claude to write social posts, executive summaries, scripts, and professional copy.

## Learning Objectives

By the end of this lecture, students should understand:

* Specify the audience, tone, and purpose before writing.
* Provide examples of the style you want.
* Use critique-and-revise loops to improve the first draft.
* Ask Claude to produce multiple versions for A/B comparison.
