# 103 - AI Agentic Frameworks

## Section

Claude Code for Building AI Agents

## Main Idea

Survey the landscape of agent frameworks and understand when to use each one.

## Learning Objectives

By the end of this lecture, students should understand:

* Multiple frameworks exist: OpenAI Agents SDK, LangGraph, AutoGen, CrewAI, and others.
* Framework choice depends on use case complexity, team expertise, and integration needs.
* The OpenAI Agents SDK is used in this course for its simplicity and power.
* Framework knowledge transfers: core agent concepts apply across all frameworks.
