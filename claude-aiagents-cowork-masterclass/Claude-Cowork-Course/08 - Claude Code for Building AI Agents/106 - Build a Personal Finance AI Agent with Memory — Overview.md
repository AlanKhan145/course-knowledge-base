# 106 - Build a Personal Finance AI Agent with Memory — Overview

## Section

Claude Code for Building AI Agents

## Main Idea

Plan a personal finance agent that can track expenses and remember financial context across sessions.

## Learning Objectives

By the end of this lecture, students should understand:

* Define the finance agent's purpose: track expenses, categorize spending, give budget advice.
* Identify what information needs to be remembered: past expenses, budget goals, spending patterns.
* Plan memory implementation: simple file-based or vector store.
* Define the agent's tool set: expense logging, category lookup, budget calculation.
