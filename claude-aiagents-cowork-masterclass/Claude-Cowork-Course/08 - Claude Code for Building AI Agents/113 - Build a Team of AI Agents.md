# 113 - Build a Team of AI Agents

## Section

Claude Code for Building AI Agents

## Main Idea

Understand multi-agent team architecture and how to coordinate specialized agents.

## Learning Objectives

By the end of this lecture, students should understand:

* A multi-agent team assigns different roles to specialized agents: researcher, analyst, writer, reviewer, coordinator.
* Agents communicate through handoffs and shared context.
* A coordinator agent manages task routing and result integration.
* Team-based agents outperform single agents on complex, multi-domain tasks.
