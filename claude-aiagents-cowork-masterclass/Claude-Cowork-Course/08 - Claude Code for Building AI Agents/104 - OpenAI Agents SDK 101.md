Claude Code for Building AI Agents

## Main Idea

Learn the OpenAI Agents SDK at a high level, covering its core concepts and how it structures agent behavior.

## Learning Objectives

By the end of this lecture, students should understand:

* The SDK provides agents, tools, handoffs, and guardrails as core primitives.
* Agents are defined with instructions, tools, and model specification.
* Handoffs enable multi-agent coordination.
* Guardrails add safety constraints to agent behavior.
