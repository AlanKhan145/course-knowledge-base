## Section

Claude Code for Building AI Agents

## Main Idea

Learn the fundamental components of an AI agent and how they interact.

## Learning Objectives

By the end of this lecture, students should understand:

* An agent consists of model, tools, instructions, memory, planning loop, evaluation, and human approval.
* Agents perceive their environment, decide on actions, and execute them.
* The planning loop is what separates agents from simple AI responses.
* Human approval gates prevent irreversible mistakes.
