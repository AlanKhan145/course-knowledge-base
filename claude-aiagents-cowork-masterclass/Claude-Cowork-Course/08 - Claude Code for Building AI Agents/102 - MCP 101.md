# 102 - MCP 101

## Section

Claude Code for Building AI Agents

## Main Idea

Understand MCP from an agent-building perspective, focusing on how to connect agents to external tools.

## Learning Objectives

By the end of this lecture, students should understand:

* MCP defines a standard interface for connecting models to tools and data sources.
* Any tool with an MCP server can be connected to any MCP-compatible agent.
* MCP makes tool integrations portable and composable.
* Building MCP-compatible tools enables reuse across multiple agent workflows.
