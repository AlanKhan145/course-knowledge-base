## Section

## Main Idea

Create and execute a simple agent using the OpenAI Agents SDK.

## Learning Objectives

By the end of this lecture, students should understand:

* Install the OpenAI Agents SDK.
* Define an agent with instructions and a model.
* Add a tool to the agent.
* Run the agent and observe the output.
* Iterate on instructions to improve behavior.
