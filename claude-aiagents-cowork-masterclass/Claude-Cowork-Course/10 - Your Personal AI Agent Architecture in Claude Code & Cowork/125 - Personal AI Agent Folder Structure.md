# 125 - Personal AI Agent Folder Structure

## Section

Your Personal AI Agent Architecture in Claude Code & Cowork

## Main Idea

Design a folder structure for your personal AI agent that is organized, extensible, and clear.

## Learning Objectives

By the end of this lecture, students should understand:

* The recommended structure includes: inbox, profile, goals, projects, brand, automations, research, outputs, CLAUDE.md, and SOUL.md.
* Folders represent domains of life and work.
* The structure should match your actual work patterns.
* Keep the structure simple enough to maintain consistently.
