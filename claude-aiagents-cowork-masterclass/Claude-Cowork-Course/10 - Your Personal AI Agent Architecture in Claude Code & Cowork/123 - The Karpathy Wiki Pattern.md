# 123 - The Karpathy Wiki Pattern

## Section

Your Personal AI Agent Architecture in Claude Code & Cowork

## Main Idea

Use a personal wiki-style knowledge base as agent context, inspired by Andrej Karpathy's approach.

## Learning Objectives

By the end of this lecture, students should understand:

* A personal wiki is a collection of markdown files about yourself, your work, and your knowledge.
* The wiki provides rich, searchable context for the agent.
* Good wiki entries are factual, concise, and regularly updated.
* The wiki becomes more valuable as it grows over time.
