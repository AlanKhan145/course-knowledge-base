## Section

Your Personal AI Agent Architecture in Claude Code & Cowork

## Main Idea

Learn the full architecture of a personal agent system including all layers and components.

## Learning Objectives

By the end of this lecture, students should understand:

* A personal agent system has seven layers: knowledge vault, personal memory, tool connectors, automation scripts, brand assets, agent playbook, and dispatch mode.
* Each layer serves a specific function in the agent's operation.
* Layers are modular: you can add or improve one without rebuilding others.
* The architecture scales from simple to complex as you add more content and tools.
