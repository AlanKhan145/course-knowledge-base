# 127 - Give Your Personal AI Agent a Personality Using SOUL.md

## Section

Your Personal AI Agent Architecture in Claude Code & Cowork

## Main Idea

Use SOUL.md to define the personality, preferences, tone, and decision style of your personal agent.

## Learning Objectives

By the end of this lecture, students should understand:

* SOUL.md contains your communication preferences, decision-making style, values, and personality traits.
* It guides how the agent communicates on your behalf.
* Good SOUL.md files are specific and honest, not aspirational.
* SOUL.md is read by the agent at the start of every session.
