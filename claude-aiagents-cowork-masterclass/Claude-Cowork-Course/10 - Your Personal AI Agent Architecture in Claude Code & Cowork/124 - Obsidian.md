## Main Idea

Use Obsidian to view and manage the personal knowledge vault visually.

## Learning Objectives

By the end of this lecture, students should understand:

* Obsidian is a markdown-based knowledge management tool with graph view.
* It can open and display the agent vault folder.
* Graph view shows connections between vault files.
* Obsidian tags and links help organize and navigate the vault.
