# 128 - Setting Up the Agent Playbook via CLAUDE.md

## Section

Your Personal AI Agent Architecture in Claude Code & Cowork

## Main Idea

Use CLAUDE.md as the operational manual for your personal agent, defining what it does and how.

## Learning Objectives

By the end of this lecture, students should understand:

* CLAUDE.md contains the agent's mission, workflow rules, tool permissions, and output standards.
* It is more operational than SOUL.md which is more identity-focused.
* Good CLAUDE.md files prevent the agent from making common mistakes.
* Update CLAUDE.md whenever you discover a recurring issue with agent behavior.
