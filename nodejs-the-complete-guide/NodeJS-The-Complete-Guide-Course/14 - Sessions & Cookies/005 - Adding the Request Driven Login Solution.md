## Key Points
* Requests carry method, path, headers, query data, and sometimes a body.
* Responses should set a meaningful status code and return the expected content type.
* Headers influence caching, content negotiation, redirects, cookies, and security.
* Authentication proves who the user is; authorization decides what that user may do.
* Passwords should be hashed with a purpose-built algorithm, never stored as plain text.
