## 1. Tom tat

Bai nay giai thich **Withdrawal Delay** trong boi canh Blockchain Developer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong blockchain, smart contract, dApp, security, oracle, scaling hoac portfolio workflow.

## 2. Muc tieu hoc tap

- Giai thich duoc Withdrawal Delay bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den trust, security, gas cost, UX, decentralization hoac deployment risk nao.
- Ap dung vao mot artifact nho: note, diagram, Solidity code, test, deploy script, dApp UI, audit finding hoac README.

## 3. Khai niem chinh

- Withdrawal Delay is part of the Rollups topic in the Blockchain Developer roadmap.
- Scaling systems move work off L1 while keeping some security relationship with L1.
- Compare fees, finality, data availability and withdrawal behavior.
- Learn it by connecting the concept to trust assumptions, transaction flow, developer workflow and security risk.
- A blockchain developer should know how to explain, test and demo this topic in a small project.

## 4. Thuc hanh

1. Draw the L1/L2 or off-chain/on-chain flow.
2. Compare fee, finality and withdrawal behavior.
3. Test or document one L2 testnet deployment step.

## 5. Bai tap

Compare L1 and one scaling approach by fee, finality, data availability and user experience.
