## 3. Khai niem chinh

- ethers.js is a JavaScript/TypeScript library for interacting with Ethereum wallets, providers and contracts.
- It is commonly used in React dApps to read chain state and send transactions.
- A good frontend handles wallet errors, wrong network, pending transactions and confirmations.
