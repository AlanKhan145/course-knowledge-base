## 1. Tom tat

Bai nay giai thich **Blockchain** trong boi canh Blockchain Developer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong blockchain, smart contract, dApp, security, oracle, scaling hoac portfolio workflow.

## 2. Muc tieu hoc tap

- Giai thich duoc Blockchain bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den trust, security, gas cost, UX, decentralization hoac deployment risk nao.
- Ap dung vao mot artifact nho: note, diagram, Solidity code, test, deploy script, dApp UI, audit finding hoac README.

## 3. Khai niem chinh

- Blockchain is part of the dApp Architecture topic in the Blockchain Developer roadmap.
- dApps are full-stack apps with an extra wallet/RPC/chain boundary.
- Design pending, failed and confirmed transaction states as first-class UI states.
- Learn it by connecting the concept to trust assumptions, transaction flow, developer workflow and security risk.
- A blockchain developer should know how to explain, test and demo this topic in a small project.

## 4. Thuc hanh

1. Build a tiny frontend or pseudocode flow for this concept.
2. Handle wallet connection, pending transaction and error state.
3. Add a screenshot or README note for the portfolio.

## 5. Bai tap

Build a small UI flow or diagram that handles wallet connect, transaction pending and failure states.
