## 3. Khai niem chinh

- An Activity is a screen-level Android component with a lifecycle managed by the system.
- It coordinates UI, navigation entry points, lifecycle callbacks and state restoration.
- Modern apps keep heavy logic out of Activity and delegate state to ViewModel or UI state holders.
