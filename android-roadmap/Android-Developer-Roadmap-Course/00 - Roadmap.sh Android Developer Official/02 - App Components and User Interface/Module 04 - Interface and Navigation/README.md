# Module 04 - Interface and Navigation

**Hoc phan:** 02 - App Components and User Interface

## Ket qua dau ra

Build Android UI with traditional layouts and modern Jetpack Compose, then connect screens with navigation.

## Project

Mini project: Shopping List App with list, form, dialog, edit/delete and detail navigation.
