## 3. Khai niem chinh

- Firebase provides common backend services such as auth, database, notifications, analytics and crash reporting.
- It helps small Android teams ship production-like features without building every backend service first.
- Treat Firebase configuration, security rules and privacy as production code.

## 4. Thuc hanh

3. Add one failure or disabled-service scenario.
