## 1. Tom tat

Bai nay giai thich **DTO** trong boi canh Android Developer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong app Android, lien quan den UI, lifecycle, state, data, network, quality hoac release nhu the nao va co the bien thanh artifact trong portfolio.

## 2. Muc tieu hoc tap

- Giai thich duoc DTO bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den UX, do on dinh, maintainability, performance hoac release risk nao.
- Ap dung vao mot app nho bang code, diagram, test, checklist, screenshot hoac README.

## 3. Khai niem chinh

- DTO is part of the HTTP Fundamentals topic in the Android Developer roadmap.
- Learn the concept by connecting API usage, lifecycle behavior, state and user impact.
- A strong Android developer can explain where this belongs in the app and how to test it.

## 4. Thuc hanh

1. Call or mock one endpoint and map the response to a UI model.
2. Implement loading, success and error states.
3. Add one retry or offline note.

## 5. Bai tap

Build or mock one API call and show loading, success, error and retry UI states.

## 6. Checklist hoan thanh

- [ ] Co dinh nghia ngan gon.

## 7. Ghi chu san xuat

Khi dua vao production, hay hoi: user flow nao bi anh huong, state co bi mat khi rotate/background khong, loi network/storage duoc xu ly ra sao, test nao bao ve tinh nang nay va release checklist co can cap nhat khong.
