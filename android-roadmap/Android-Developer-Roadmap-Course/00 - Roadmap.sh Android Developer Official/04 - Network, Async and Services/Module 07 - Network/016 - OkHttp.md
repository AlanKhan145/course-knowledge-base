## 3. Khai niem chinh

- OkHttp is the HTTP client commonly used under Retrofit.
- Interceptors can add headers, logging, authentication, caching and retry behavior.
- Be deliberate with timeouts and avoid logging sensitive data.

## 4. Thuc hanh
