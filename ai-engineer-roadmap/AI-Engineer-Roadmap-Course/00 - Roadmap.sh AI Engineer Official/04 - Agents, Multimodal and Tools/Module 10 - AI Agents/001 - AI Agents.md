## 3. Khai niem chinh

- AI agents use an LLM to plan, call tools, inspect results and decide next steps.
- Agents are powerful when tasks need multi-step interaction with APIs, files, search or databases.
- They also need permission boundaries, logging, human approval and failure handling.

## 4. Vi du / Demo

```text
User goal -> planner -> tool call -> tool result -> next action -> final response
```

## 5. Bai tap thuc hanh

- Dinh nghia mot tool nho co schema ro rang.
- Cho agent giai mot task 2-3 buoc va log tung tool call.
- Them permission boundary va stop condition.

## 6. Loi thuong gap

- Cho agent qua nhieu quyen hoac tool nguy hiem.
- Khong log intermediate steps nen khong debug duoc.
- Khong co timeout, budget, stop condition va human approval.
