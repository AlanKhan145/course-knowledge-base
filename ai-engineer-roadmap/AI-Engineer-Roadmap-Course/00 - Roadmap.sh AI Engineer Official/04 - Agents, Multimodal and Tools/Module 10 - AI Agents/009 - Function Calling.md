## 3. Khai niem chinh

- Function calling lets an agent request a typed operation such as search, database lookup or email draft.
- Arguments must be validated before execution.
- Tool results should be logged and bounded so the agent cannot act outside its scope.

## 4. Vi du / Demo

```text
goal -> plan -> choose tool -> execute -> observe -> decide -> final answer
```

## 5. Bai tap thuc hanh

- Dinh nghia mot tool nho co schema ro rang.
- Cho agent giai mot task 2-3 buoc va log tung tool call.
- Them permission boundary va stop condition.

## 6. Loi thuong gap

- Cho agent qua nhieu quyen hoac tool nguy hiem.
- Khong log intermediate steps nen khong debug duoc.
- Khong co timeout, budget, stop condition va human approval.
