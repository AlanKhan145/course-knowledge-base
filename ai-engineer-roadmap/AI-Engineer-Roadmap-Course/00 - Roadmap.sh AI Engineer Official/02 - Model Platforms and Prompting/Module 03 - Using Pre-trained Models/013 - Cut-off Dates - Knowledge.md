## 3. Khai niem chinh

- A model knowledge cutoff is the latest period reflected in its training data.
- For current or private information, use tools, retrieval, APIs or browsing systems instead of relying on model memory.
- Always tell users when an answer depends on retrieved data versus model prior knowledge.
