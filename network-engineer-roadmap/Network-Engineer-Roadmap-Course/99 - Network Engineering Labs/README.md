# Network Engineering Labs

Các lab này là phần thực hành bổ sung để biến Network Engineer Roadmap thành portfolio có thể review.

- [Lab 01 - Small Office Network Design](Lab 01 - Small Office Network Design.md)
- [Lab 02 - Subnetting and Routing Lab](Lab 02 - Subnetting and Routing Lab.md)
- [Lab 03 - Switching and Wireless Segmentation Lab](Lab 03 - Switching and Wireless Segmentation Lab.md)
- [Lab 04 - Network Security Baseline](Lab 04 - Network Security Baseline.md)
- [Lab 05 - Network Observability and Troubleshooting Playbook](Lab 05 - Network Observability and Troubleshooting Playbook.md)

## Cách chọn lab
- Nếu bạn mới bắt đầu: chọn Lab 01.
- Nếu muốn luyện subnet/routing: chọn Lab 02.
- Nếu muốn luyện switching/wireless: chọn Lab 03.
- Nếu muốn đi security: chọn Lab 04.
- Nếu muốn đi operations/NOC/SRE networking: chọn Lab 05.
