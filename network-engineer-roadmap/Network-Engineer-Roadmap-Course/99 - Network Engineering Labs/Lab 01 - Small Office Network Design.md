# Lab 01 - Small Office Network Design

## Bối cảnh
Thiết kế mạng cho văn phòng nhỏ có wired LAN, Wi-Fi, guest network, Internet gateway và nhu cầu truy cập dịch vụ nội bộ.

## Mục tiêu
- Chứng minh bạn hiểu topology, addressing, routing/switching, security hoặc observability ở mức thực hành.
- Tạo artifact có thể đưa vào portfolio Network Engineer.
- Ghi lại trade-off, giả định, failure mode và cách kiểm chứng.

## Deliverable chính
- Topology diagram
- IP addressing plan
- VLAN design
- Device role explanation
- Basic security baseline

## Công cụ gợi ý
- Diagram tool
- Subnet calculator
- Router/switch simulator nếu có
- README design note

## Yêu cầu chất lượng
- Có sơ đồ hoặc bảng thiết kế rõ ràng.
- Có IP/subnet/protocol/device role nếu phù hợp.
- Có phần troubleshooting hoặc validation plan.
- Có rủi ro bảo mật và vận hành.
- Có README giải thích quyết định kỹ thuật.

## Câu hỏi review
- Packet đi từ đâu tới đâu, qua thiết bị nào?
- Nếu một link/device/service lỗi, hệ thống phản ứng ra sao?
- Security baseline có giảm rủi ro rõ ràng không?
- Tool nào dùng để xác nhận thiết kế hoạt động đúng?
