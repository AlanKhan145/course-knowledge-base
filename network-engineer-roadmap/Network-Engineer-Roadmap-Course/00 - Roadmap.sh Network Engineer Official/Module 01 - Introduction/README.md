# Module 01 - Introduction

## Mục tiêu module
Hiểu Internet hoạt động ở mức tổng quan: packet, DNS, IP, TCP/UDP và HTTP/HTTPS.

## Bài học

- Lab note hoặc troubleshooting checklist có thể dùng lại
- Danh sách rủi ro, câu hỏi mở hoặc command/tool cần luyện thêm
