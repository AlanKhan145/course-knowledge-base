# Learning Path

Lộ trình học đề xuất được chuẩn hóa từ nguồn để giúp bạn đi từ nền tảng Internet tới security và monitoring.

## Giai đoạn 1 - Nền tảng mạng

### Nội dung học
- Học Internet hoạt động như thế nào, network types, terminology, core protocols.

### Artifact nên tạo
- Nền tảng mạng checklist
- Một lab note hoặc sơ đồ mạng gắn với giai đoạn này
- Danh sách command/tool cần luyện thêm
