# Cyber Security Expert Roadmap 2026

Khóa học markdown được tạo từ roadmap PDF và bản brief tiếng Việt đã chuẩn hóa.

## Mục tiêu khóa học

- Nắm nền tảng IT, hệ điều hành, mạng máy tính và bảo mật
- Hiểu các kỹ năng cốt lõi của Cyber Security: phòng thủ, tấn công, điều tra sự cố, mã hóa, cloud security
- Làm quen với CTF, lab hacking hợp pháp và các công cụ phổ biến
- Có lộ trình học chứng chỉ từ cơ bản đến nâng cao
- Xây dựng portfolio gồm lab, write-up CTF, báo cáo phân tích log, báo cáo incident response và cloud security checklist

## Thống kê

- Phần: 19
- Module: 19
- Bài học/project: 114
- Ngôn ngữ: Vietnamese
- Format: Markdown lesson notes, module README, course index, syllabus và coverage report

## Cách học

1. Mở `COURSE_INDEX.md` để xem toàn bộ lộ trình.
2. Học theo thứ tự phần/module/bài.
3. Với mỗi bài, tạo một artifact nhỏ: checklist, command note, script, lab report, diagram hoặc runbook.
4. Cuối khóa, hoàn thành 6 project để xây portfolio.
