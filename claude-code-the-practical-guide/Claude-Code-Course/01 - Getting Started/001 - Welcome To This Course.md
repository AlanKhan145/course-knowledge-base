# 001 - Welcome To This Course

## Section

Getting Started

## Main Idea

This lecture introduces the Claude Code - The Practical Guide course. The instructor, Maximilian Schwarzmüller from Academind, welcomes learners and explains why Claude Code is more than just a chatbot or simple code generator.

Claude Code is presented as a serious AI coding assistant that can inspect project files, edit code, run terminal commands, and support advanced agentic engineering workflows. The course is designed to take developers from basic usage all the way to complex automated workflows.

## Learning Objectives

By the end of this lecture, students should understand:

* What Claude Code is.
* Why Claude Code is useful for developers.
* What the course will cover.
* Who this course is designed for.
* What kind of workflows Claude Code can support.

## Key Concepts

### Claude Code

Claude Code is an AI coding assistant built by Anthropic. It runs in the terminal, integrates with IDEs, and can perform real development tasks such as reading files, editing code, running commands, and managing project context.

### AI Coding Assistant vs Simple Chatbot

Unlike a basic AI chatbot that only generates text responses, Claude Code actively works inside your project. It can inspect the codebase, make targeted edits, run commands, and integrate with external tools.

### Agentic Workflows

Claude Code supports agentic workflows where the AI takes sequences of actions, evaluates results, and iterates. This is more powerful than single-turn question-and-answer interactions.

### Course Philosophy

The course is practical and project-focused. Students will learn by applying Claude Code to real development tasks, not just by reading theory.

## Important Notes

* A Claude Code subscription is required to follow the course.
* The course targets beginner to intermediate developers.
* No prior Claude Code experience is needed.
* Basic programming knowledge is assumed.
* The course covers both simple usage and advanced agentic workflows.
