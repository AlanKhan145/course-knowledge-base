# 018 - Course Resources & Community

## Section

Getting Started

## Main Idea

This lecture explains where to find course resources, code examples, and community support. Knowing these resources before diving into the main content helps students get unstuck faster and learn more effectively.

## Learning Objectives

By the end of this lecture, students should understand:

* Where to find course code examples and resources.
* How to access the course community for help.
* What to do when stuck on a concept or technical issue.
* How to give feedback on the course.

## Key Concepts

### Course Code Resources

The course provides code snapshots and demo projects that correspond to specific lectures. These resources allow students to:

* Compare their work against the reference implementation.
* Start from a known good state if they fall behind.
* Download finished versions of demo projects.

Resources are typically linked in the Udemy course page under each lecture's resources section.

### Community Support

**Udemy Q&A:**
Every lecture has a Q&A section where students can ask questions. The instructor and community members can answer. This is the primary channel for course-specific questions.

**Course Discussion Board:**
A broader discussion area for general questions about Claude Code and the course.

### Official Claude Code Resources

For questions beyond the course:

* **Anthropic Documentation:** The official Claude Code documentation at Anthropic's website covers all features, APIs, and configuration options.
* **Anthropic Discord:** The Anthropic community Discord is a good place to ask questions, share workflows, and learn from other Claude Code users.
* **GitHub Issues:** For bugs and feature requests in Claude Code, the official GitHub repository tracks issues and updates.

### Getting Help Effectively

When asking for help:

1. Describe what you expected to happen.
2. Describe what actually happened.
3. Share the error message or output if there is one.
4. Share the relevant code or configuration.
5. Mention your operating system and Claude Code version.

Specific questions get faster and more accurate answers than vague ones.

### Staying Current

Claude Code is actively developed and features change. To stay current:

* Watch the Anthropic blog for announcements.
* Follow the Claude Code changelog.
* Check the course Q&A for notes about changes that affect course content.

## Important Notes

* Claude Code is updated frequently. Some course details may differ from the latest version.
* If a command or feature does not work as shown, check whether it has been updated in newer Claude Code versions.
* The Udemy Q&A is monitored but response times vary.
* Community members are often very helpful for practical questions and workflow tips.

## Why This Lecture Matters

Students who know where to find help get unstuck faster. This lecture also sets the expectation that Claude Code changes over time and that checking current documentation is part of the workflow.

## Summary

Course resources including code snapshots are available through the Udemy course page. Community support is available through the Udemy Q&A, the Anthropic Discord, and official documentation. When asking for help, specific questions with clear context get the best responses. Because Claude Code is updated frequently, always cross-reference with current documentation when features differ from the course content.
