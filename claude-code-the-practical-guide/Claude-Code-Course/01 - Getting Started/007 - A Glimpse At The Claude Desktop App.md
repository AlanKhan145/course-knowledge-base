# 007 - A Glimpse At The Claude Desktop App

## Section

Getting Started

## Main Idea

This lecture introduces the Claude Desktop app and explains how it differs from the CLI experience. Students get a quick preview of the Desktop app so they understand the landscape of tools available before diving deeper into the CLI workflow.

## Learning Objectives

By the end of this lecture, students should understand:

* What the Claude Desktop app is.
* How the Desktop app differs from the Claude Code CLI.
* When to use the Desktop app versus the CLI.
* What capabilities the Desktop app offers.

## Key Concepts

### Claude Desktop App

The Claude Desktop app is a native application for macOS and Windows that provides a graphical interface for interacting with Claude. It offers features beyond what the basic web chat interface provides.

### Desktop App vs CLI

| Feature | Claude Code CLI | Claude Desktop App |
|---|---|---|
| Terminal access | Yes | Limited |
| File system access | Full project access | Configured access |
| IDE integration | Yes | No |
| Visual interface | No | Yes |
| Good for planning | No | Yes |
| Good for coding | Yes | Limited |
| MCP server support | Yes | Yes |

### When to Use the Desktop App

The Desktop app is useful for:

* High-level project planning and discussion.
* Reviewing and summarizing documents.
* Working with Claude on tasks that are not code-focused.
* Using connected integrations (MCP servers) in a visual interface.
* Drafting communications or documentation.

### When to Use the CLI

The CLI is better for:

* Direct coding tasks.
* File editing and inspection.
* Running commands.
* IDE-integrated workflows.
* Complex agentic sequences.

### MCP Servers in Desktop

The Desktop app also supports MCP servers, which means external tools like GitHub, databases, or browsers can be connected to Claude through the Desktop interface as well.

## Important Notes

* The Desktop app and CLI are complementary tools, not competitors.
* Some workflows benefit from using both: plan in the Desktop app, implement in the CLI.
* Section 3 of the course covers the Desktop app in detail.
* This lecture is just an introduction to avoid confusion about what the Desktop app is.

## Why This Lecture Matters

Many students only know about the web chat interface for Claude. Introducing the Desktop app early helps students understand the full ecosystem of Claude tools and prevents confusion when the Desktop app is mentioned later in the course.

## Summary

The Claude Desktop app is a native application that offers a visual interface for Claude. It complements the CLI by being better for planning, document work, and visual workflows, while the CLI is better for direct coding and file manipulation. The course covers the Desktop app in detail in Section 3.
