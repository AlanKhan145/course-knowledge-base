# 002 - Course Overview

## Section

Getting Started

## Main Idea

This lecture gives a structured overview of the entire course. It explains how the lessons are organized across three sections and what each section covers. Students will know what to expect at every stage of the course before they begin.

The course is organized from foundational setup through advanced productivity features and finally beyond the local CLI into web, desktop, mobile, and scheduled workflow usage.

## Learning Objectives

By the end of this lecture, students should understand:

* The three main sections of the course.
* What topics are covered in each section.
* How the course builds progressively from basic to advanced.
* Which features require prior concepts to be understood first.
* What the final outcome of the course is.

## Key Concepts

### Section 1: Getting Started

Covers installation, configuration, basic usage, sessions, context, permissions, sandboxing, and version control. These are the foundations every student needs before using advanced features.

### Section 2: Key Features & Efficient Usage

Covers the core productivity features: prompt engineering, context engineering, specs, `CLAUDE.md`, Plan Mode, built-in tools, MCP servers, subagents, skills, custom commands, hooks, plugins, browser feedback loops, automated tests, and agent loops.

### Section 3: Beyond Local CLI Usage

Covers Claude Code outside the terminal: Desktop app, mobile task dispatch, remote control, Telegram channels, non-code use cases, and scheduled tasks.

### Progressive Learning

Each concept in the course builds on what came before. Students should work through the sections in order rather than jumping ahead.

## Important Notes

* The course covers ~54 lectures across 3 sections.
* Total duration is approximately 3 hours.
* The course is practical and demo-focused.
* Students should follow along with their own project or the provided demo app.
* Advanced topics like MCP servers and agent loops require the basic skills to be solid first.

## Course Map

| Section | Focus |
|---|---|
| Section 1 | Setup, configuration, sessions, permissions, safety |
| Section 2 | Prompts, specs, CLAUDE.md, Plan Mode, tools, MCP, subagents, skills, hooks, feedback loops |
| Section 3 | Web, desktop, mobile, remote, scheduled tasks |

## Why This Lecture Matters

Knowing the course structure before starting helps students plan their study time, understand which concepts are prerequisites for others, and avoid confusion when a feature depends on something covered earlier.

## Summary

This lecture maps out the full course across three sections: Getting Started, Key Features & Efficient Usage, and Beyond Local CLI Usage. The course follows a progressive structure, starting from installation and moving to advanced agentic workflows and non-local usage. Students should follow the course in order to build the right foundation before tackling the more advanced features.
