## 1. Tom tat

Bai nay giai thich **XCTest** trong boi canh iOS Developer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong app iOS, lien quan den Swift, UI, lifecycle, architecture, data, networking, testing hoac distribution nhu the nao.

## 2. Muc tieu hoc tap

- Giai thich duoc XCTest bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den UX, maintainability, performance, testability hoac release risk nao.
- Ap dung vao mot artifact nho: Swift snippet, UI screen, test, checklist, diagram, screenshot hoac README.

## 3. Khai niem chinh

- XCTest is Apple's testing framework for unit tests and integration-style tests.
- Use it for ViewModels, services, parsing, repositories and pure Swift logic.
- Tests are most valuable when they protect state transitions and business rules.

## 4. Thuc hanh

1. Write one focused test for a state transition or user flow.
2. Use a fake or mock for external dependencies.
3. Record the test command and expected result.

## 5. Bai tap

Write one repeatable test and explain what regression it protects against.
