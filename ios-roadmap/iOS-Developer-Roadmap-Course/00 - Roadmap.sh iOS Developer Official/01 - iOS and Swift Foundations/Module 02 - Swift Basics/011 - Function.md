## 3. Khai niem chinh

- Function is part of the Swift Fundamentals topic in the iOS Developer roadmap.
- Practice this in small Swift files before mixing in UI framework complexity.
- Connect each syntax topic to app code such as models, view state, services and errors.

- An iOS developer should know where this concept belongs in a real app and how to test it.
