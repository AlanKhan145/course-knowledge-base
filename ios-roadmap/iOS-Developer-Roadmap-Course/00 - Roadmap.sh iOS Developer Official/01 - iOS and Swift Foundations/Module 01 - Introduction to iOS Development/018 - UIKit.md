## 1. Tom tat

Bai nay giai thich **UIKit** trong boi canh iOS Developer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong app iOS, lien quan den Swift, UI, lifecycle, architecture, data, networking, testing hoac distribution nhu the nao.

## 2. Muc tieu hoc tap

- Giai thich duoc UIKit bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den UX, maintainability, performance, testability hoac release risk nao.
- Ap dung vao mot artifact nho: Swift snippet, UI screen, test, checklist, diagram, screenshot hoac README.

## 3. Khai niem chinh

- UIKit is Apple's long-standing imperative UI framework for iOS.
- Many production apps still use UIKit, so understanding UIViewController, Auto Layout and navigation remains valuable.
- UIKit knowledge helps maintain legacy code and bridge UIKit components into SwiftUI when needed.
