## 3. Khai niem chinh

- @Binding is part of the SwiftUI State topic in the iOS Developer roadmap.
- SwiftUI bugs often come from state ownership confusion.
- Decide whether state is local, bound, observable or environment-level before writing the view.
- Learn it by connecting the API, state, lifecycle, user experience and release impact.
- An iOS developer should know where this concept belongs in a real app and how to test it.
- Decide who owns the state and how changes flow back into the UI.
