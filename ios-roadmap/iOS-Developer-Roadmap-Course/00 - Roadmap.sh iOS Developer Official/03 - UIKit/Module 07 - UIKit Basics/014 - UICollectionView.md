## 3. Khai niem chinh

- UICollectionView is part of the UIKit Foundation topic in the iOS Developer roadmap.
- UIKit is lifecycle-driven, so log lifecycle callbacks and test navigation behavior.
- Keep business logic out of view controllers when possible.
- Learn it by connecting the API, state, lifecycle, user experience and release impact.
- An iOS developer should know where this concept belongs in a real app and how to test it.
