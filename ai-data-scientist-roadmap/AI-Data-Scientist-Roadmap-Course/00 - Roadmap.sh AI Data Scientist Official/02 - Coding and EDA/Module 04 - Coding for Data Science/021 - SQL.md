## 3. Khai niem chinh

- SQL is essential for extracting and aggregating data from databases.
- Learn joins, group by, subqueries, CTEs and window functions.
- Always know the grain of your query result to avoid duplicate or misleading metrics.

## 4. Vi du / Demo

```sql
SELECT
  DATE_TRUNC('month', order_date) AS month,
  SUM(revenue) AS revenue
FROM orders
GROUP BY 1
ORDER BY 1;
```
