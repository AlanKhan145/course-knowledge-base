## 2. Muc tieu hoc tap

- Giai thich duoc Feature Engineering bang ngon ngu cua ban.
- Nhan biet topic nay nam o dau trong workflow AI/Data Scientist.
- Ap dung vao mot dataset, notebook, model, experiment, dashboard hoac deployment artifact nho.

## 3. Khai niem chinh

- Feature engineering transforms raw data into useful model inputs.
- Examples include scaling, encoding, time features, bins, interactions and text features.
- Feature leakage is a common source of falsely high performance.

## 4. Vi du / Demo

```text
raw fields -> cleaned features -> encoded/scaled features -> train/validation split
```

## 5. Bai tap thuc hanh

- Train baseline va it nhat mot model khac.
- So sanh metric tren validation/test set.
- Ghi lai error analysis va next feature can thu.

## 6. Loi thuong gap

- Data leakage giua train va test.
- Chon metric sai voi business problem.
- Dung model phuc tap ma khong co baseline.
