## 1. Tom tat

Bai nay giai thich **Confusion Matrix** trong boi canh AI & Data Scientist. Sau bai hoc, ban nen biet topic nay giup tra loi cau hoi data nao, lien quan den model/experiment/deployment nao va co the bien thanh notebook, metric, chart, API hoac portfolio artifact.

## 2. Muc tieu hoc tap

- Giai thich duoc Confusion Matrix bang ngon ngu cua ban.
- Nhan biet topic nay nam o dau trong workflow AI/Data Scientist.
- Ap dung vao mot dataset, notebook, model, experiment, dashboard hoac deployment artifact nho.

## 3. Khai niem chinh

- Confusion Matrix la mot phan cua workflow train va chon model ML.
- Can so sanh voi baseline, dung metric phu hop va tranh data leakage.
- Model tot phai giai quyet cau hoi business, khong chi co score cao.

## 4. Vi du / Demo

```text
data -> split -> features -> baseline -> model -> metric -> error analysis
```

## 5. Bai tap thuc hanh

- Train baseline va it nhat mot model khac.
- So sanh metric tren validation/test set.
- Ghi lai error analysis va next feature can thu.

## 6. Loi thuong gap

- Data leakage giua train va test.
- Chon metric sai voi business problem.
- Dung model phuc tap ma khong co baseline.
