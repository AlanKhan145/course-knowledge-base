## 1. Tom tat

Bai nay giai thich **Data Analyst to Data Scientist** trong boi canh AI & Data Scientist. Sau bai hoc, ban nen biet topic nay giup tra loi cau hoi data nao, lien quan den model/experiment/deployment nao va co the bien thanh notebook, metric, chart, API hoac portfolio artifact.

## 2. Muc tieu hoc tap

- Giai thich duoc Data Analyst to Data Scientist bang ngon ngu cua ban.
- Nhan biet topic nay nam o dau trong workflow AI/Data Scientist.
- Ap dung vao mot dataset, notebook, model, experiment, dashboard hoac deployment artifact nho.

## 3. Khai niem chinh

- Data Analyst to Data Scientist giup dong goi nang luc thanh bang chung co the review.
- Portfolio can co folder ro, README, data note, notebooks, code, model va cach chay.
- Muc tieu la cho thay ban co the di tu raw data den insight/model/deployment.

## 4. Vi du / Demo

```text
Input: mot dataset hoac bai toan nho
Process: ap dung topic vao analysis, model, chart, test hoac deployment
Output: notebook, metric, insight, caveat va next step
```

## 5. Bai tap thuc hanh

- Tao README co problem, data, method, result va limitation.
- Sap xep folder de nguoi khac doc/chay lai duoc.
- Them chart, metric va recommendation cuoi cung.

## 6. Loi thuong gap

- Hoc thuoc dinh nghia nhung khong tao artifact thuc hanh.
- Khong ghi lai assumption, limitation va caveat.
- Bo qua validation vi demo nho co ve chay dung.

## 7. Checklist hoan thanh

- Toi co the giai thich **Data Analyst to Data Scientist** trong 1-2 phut.
- Toi co mot notebook, query, chart, model, API hoac ghi chu thuc hanh cho bai nay.
- Toi biet topic nay lien quan den dataset, metric, model, experiment hoac deployment nao.
- Toi da ghi lai it nhat mot caveat, assumption hoac cau hoi can phan tich tiep.

## 8. Outcome lien quan

Organize projects so reviewers can run, inspect and trust your work.

## 9. Project lien quan

Create a portfolio repository with consistent folder structure and README files.

## 10. Tong ket

**Data Analyst to Data Scientist** la mot moc trong lo trinh AI & Data Scientist. Hay bien no thanh mot notebook, query, chart, experiment, model, API, Docker service hoac portfolio note de kien thuc co cho bam.
