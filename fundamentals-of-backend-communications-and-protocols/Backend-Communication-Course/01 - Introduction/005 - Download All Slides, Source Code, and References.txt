# 005 - Download All Slides, Source Code, and References

**Section:** 01 - Introduction
**Duration:** 1min

---

## 1. Summary

This short lesson points students to the location where all course resources are stored. The resources may include slides, source code, references, and other supporting materials needed throughout the course.

The instructor wants students to download or keep these files in one organized place so they can easily access them while following the lessons.

Resource folder path:

```text id="sf5isg"
D:\Sao lưu\Udemy\fundamentals-of-backend-communications-and-protocols\Resources
```

---

## 2. Key Concepts

### Course Resources

The course resources are supporting files that help students follow the lectures more effectively.

They may include:

* Lecture slides
* Source code
* Reference files
* Diagrams
* Extra learning materials

---

### Centralized Resource Folder

Keeping all resources in one folder makes the course easier to follow. Instead of searching for files in different locations, students can open one folder whenever they need course materials.

---

### Slides

Slides are useful for reviewing concepts, diagrams, definitions, and course explanations after watching the video lectures.

---

### Source Code

Source code allows students to inspect examples, run experiments, and understand how backend communication concepts are implemented in practice.

---

### References

References provide extra materials or links that can help students study topics more deeply outside the main lectures.

---

## 3. Important Notes

* Download or locate all course resources before continuing with the main sections.
* Keep the resources organized in a dedicated folder.
* Do not delete or move the folder unless you also update your notes with the new location.
* Slides are useful for review, but they should be studied together with the video explanations.
* Source code should be opened, read, and tested when possible.
* References can help reinforce difficult backend and networking concepts.
* The path shown is a local Windows folder path.
