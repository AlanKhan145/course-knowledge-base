# 2.1 - Learn HTML Basics

## Section
Phase 2 - Core Web Languages — Module 02: HTML

## Learning Goals

- DOCTYPE, html, head, body
- meta, title, link, script
- h1-h6 headings
- p, span, strong, em, small, mark
- ul, ol, li
- a links: relative, absolute, anchor
- img alt width height
- table, thead, tbody, tr, th, td

## You Should Be Able To

- Explain learn html basics in plain English.
- Use the concept in a small frontend project.
- Debug the most common beginner mistakes around this topic.

## Practice

1. Create a CV page using only HTML.
2. Create a blog article page using only HTML.

## Mastery Checklist

- [ ] I can define the core terms without looking them up.
- [ ] I can build or inspect a real example.
- [ ] I can explain the trade-offs to another beginner.
- [ ] I can connect this topic to the next lesson in the roadmap.
