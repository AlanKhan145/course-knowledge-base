# 1.4 - What is Hosting

## Section
Phase 1 - Web Foundation — Module 01: Internet

## Learning Goals

- Hosting basics
- Static, dynamic, shared, VPS, cloud, serverless, CDN, and edge hosting
- Deploying a static frontend site
- Deploying an SSR app
- Build command and output directory

## You Should Be Able To

- Explain what is hosting in plain English.
- Use the concept in a small frontend project.
- Debug the most common beginner mistakes around this topic.
