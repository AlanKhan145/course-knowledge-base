# 1.5 - DNS and How It Works

## Section
Phase 1 - Web Foundation — Module 01: Internet

## Learning Goals

- DNS resolver
- Root DNS
- TLD nameserver
- Authoritative nameserver
- A, AAAA, CNAME, MX, TXT, and NS records
- TTL
- DNS propagation

## You Should Be Able To

- Explain dns and how it works in plain English.
- Use the concept in a small frontend project.
- Debug the most common beginner mistakes around this topic.

## Practice

1. Use nslookup or a web DNS checker.
2. Check the A record of a domain.
3. Point a custom domain to a hosting provider.

- [ ] I can define the core terms without looking them up.
