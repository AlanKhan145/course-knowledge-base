## 1. Tóm tắt
Bài này tập trung vào **Python** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Biết **Python** phù hợp với nhánh game development nào.
- Nhận biết điểm mạnh của ngôn ngữ trong gameplay, engine, tooling hoặc AI prototyping.
- Tạo được bài tập nhỏ liên quan tới game loop, input, data hoặc automation.
