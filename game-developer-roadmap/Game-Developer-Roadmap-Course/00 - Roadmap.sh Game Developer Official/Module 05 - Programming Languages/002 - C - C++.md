## 1. Tóm tắt
Bài này tập trung vào **C/C++** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Biết **C/C++** phù hợp với nhánh game development nào.
- Nhận biết điểm mạnh của ngôn ngữ trong gameplay, engine, tooling hoặc AI prototyping.
- Tạo được bài tập nhỏ liên quan tới game loop, input, data hoặc automation.

## 3. Nội dung roadmap
- C/C++

### Phù hợp với

- Unreal Engine
- Game engine
- Graphics programming
- Performance-critical systems

## 4. Bài tập thực hành
- Viết mini game loop hoặc tool script bằng ngôn ngữ tương ứng.
- Ghi lại điểm mạnh/yếu khi dùng ngôn ngữ đó cho game development.
- So sánh với một ngôn ngữ khác trong roadmap.

## 5. Artifact nên tạo
- Mini script/game loop
- Language comparison note
- Tooling snippet

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **C/C++** bằng ví dụ trong một game cụ thể không?
- Chủ đề này ảnh hưởng đến gameplay, engine, graphics, physics, AI hay build pipeline?
- Demo hoặc ghi chú nào từ bài này có thể đưa vào portfolio game developer?

## 7. Tổng kết
**C/C++** nên được học bằng cách làm prototype, quan sát lỗi và ghi lại trade-off kỹ thuật. Hãy biến bài học thành một demo nhỏ hoặc README rõ ràng.
