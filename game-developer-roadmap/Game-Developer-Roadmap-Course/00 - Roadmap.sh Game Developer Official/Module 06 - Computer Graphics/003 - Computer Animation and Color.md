# 003 - Computer Animation & Color

**Module:** Module 06 - Computer Graphics
**Roadmap item:** 6.3
**Nhóm nội dung:** Computer graphics
**Thứ tự trong module:** 003
**Thời lượng gợi ý:** 45-60 phút

---

## 1. Tóm tắt
Bài này tập trung vào **Computer Animation & Color** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Hiểu **Computer Animation & Color** trong pipeline render và asset visual của game.
- Biết liên hệ shader, texture, lighting, material hoặc animation với trải nghiệm người chơi.
- Tạo scene hoặc shader nhỏ để quan sát hiệu ứng trực tiếp.

## 3. Nội dung roadmap
### Chủ đề

- Computer Animation
- Color
- Visual Perception
- Tone Reproduction

### Cần học

- Keyframe animation
- Skeletal animation
- Skinning
- Animation blending
- Color space
- Gamma correction
- HDR
- Tone mapping

## 4. Bài tập thực hành
- Tạo scene với mesh, texture, material, light và shader đơn giản.
- Chụp before/after khi thay đổi shader, texture, normal map hoặc tone mapping.
- Viết note giải thích vì sao visual thay đổi.

## 5. Artifact nên tạo
- Shader/material demo
- Scene screenshot
- Rendering pipeline note

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Computer Animation & Color** bằng ví dụ trong một game cụ thể không?
- Chủ đề này ảnh hưởng đến gameplay, engine, graphics, physics, AI hay build pipeline?
- Demo hoặc ghi chú nào từ bài này có thể đưa vào portfolio game developer?

## 7. Tổng kết
**Computer Animation & Color** nên được học bằng cách làm prototype, quan sát lỗi và ghi lại trade-off kỹ thuật. Hãy biến bài học thành một demo nhỏ hoặc README rõ ràng.
