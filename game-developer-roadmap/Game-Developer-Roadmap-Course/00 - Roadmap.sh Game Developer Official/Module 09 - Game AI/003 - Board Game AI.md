# 003 - Board Game AI

**Module:** Module 09 - Game AI
**Roadmap item:** 9.3
**Nhóm nội dung:** Game AI
**Thứ tự trong module:** 003
**Thời lượng gợi ý:** 40-55 phút

---

## 1. Tóm tắt
Bài này tập trung vào **Board Game AI** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Dùng **Board Game AI** để thiết kế hành vi NPC, enemy, bot hoặc gameplay system.
- Biết chọn rule-based AI, search AI hoặc learning-based AI theo độ phức tạp game.
- Tạo được prototype nhỏ có state, pathfinding, decision hoặc adaptation.

## 3. Nội dung roadmap
### Chủ đề

- Minimax
- Alpha-Beta Pruning
- MCTS

### Ứng dụng

- Cờ caro
- Cờ vua mini
- Card game
- Turn-based strategy
- Puzzle game

## 4. Bài tập thực hành
- Tạo enemy AI mini với patrol, chase, attack hoặc flee.
- Biểu diễn decision logic bằng state machine, behavior tree hoặc flowchart.
- Test ít nhất ba tình huống để xem AI có hành vi mong muốn không.

## 5. Artifact nên tạo
- AI behavior prototype
- State machine/behavior tree diagram
- Test cases

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Board Game AI** bằng ví dụ trong một game cụ thể không?
- Chủ đề này ảnh hưởng đến gameplay, engine, graphics, physics, AI hay build pipeline?
- Demo hoặc ghi chú nào từ bài này có thể đưa vào portfolio game developer?

## 7. Tổng kết
**Board Game AI** nên được học bằng cách làm prototype, quan sát lỗi và ghi lại trade-off kỹ thuật. Hãy biến bài học thành một demo nhỏ hoặc README rõ ràng.
