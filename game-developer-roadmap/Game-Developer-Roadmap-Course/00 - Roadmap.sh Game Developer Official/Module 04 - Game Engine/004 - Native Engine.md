# 004 - Native Engine

**Module:** Module 04 - Game Engine
**Roadmap item:** 4.4
**Nhóm nội dung:** Game engine
**Thứ tự trong module:** 004
**Thời lượng gợi ý:** 45-60 phút

---

## 1. Tóm tắt
Bài này tập trung vào **Native Engine** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Nắm workflow cơ bản của **Native Engine** và vai trò của scene/object/component/script.
- Biết tạo prototype nhỏ trong engine hoặc native stack phù hợp.
- Hiểu trade-off giữa engine có sẵn và tự viết engine.

## 3. Nội dung roadmap
### Cần học

- C/C++
- Game loop
- Window management
- Input handling
- Renderer
- Asset loading
- Physics integration
- Audio
- ECS architecture

### Dự án thực hành

- Tự viết game engine 2D nhỏ
- Renderer OpenGL cơ bản
- Entity Component System mini

## 4. Bài tập thực hành
- Dựng prototype mini theo engine đang học, có input, object, collision và UI tối thiểu.
- Ghi lại folder structure, asset pipeline và cách build game.
- Viết README mô tả controls, gameplay loop và bài học kỹ thuật.

## 5. Artifact nên tạo
- Playable prototype
- Build README
- Engine workflow notes

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Native Engine** bằng ví dụ trong một game cụ thể không?
- Chủ đề này ảnh hưởng đến gameplay, engine, graphics, physics, AI hay build pipeline?
- Demo hoặc ghi chú nào từ bài này có thể đưa vào portfolio game developer?

## 7. Tổng kết
**Native Engine** nên được học bằng cách làm prototype, quan sát lỗi và ghi lại trade-off kỹ thuật. Hãy biến bài học thành một demo nhỏ hoặc README rõ ràng.
