# 001 - Machine Learning cho Game AI

**Module:** Module 10 - Machine Learning cho Game AI
**Roadmap item:** 10.1
**Nhóm nội dung:** Game AI
**Thứ tự trong module:** 001
**Thời lượng gợi ý:** 40-55 phút

---

## 1. Tóm tắt
Bài này tập trung vào **Machine Learning cho Game AI** trong lộ trình Game Developer. Sau bài học, bạn nên hiểu khái niệm, biết nó xuất hiện ở đâu trong game thực tế và tạo được một demo hoặc ghi chú kỹ thuật nhỏ.

## 2. Mục tiêu học tập
- Dùng **Machine Learning cho Game AI** để thiết kế hành vi NPC, enemy, bot hoặc gameplay system.
- Biết chọn rule-based AI, search AI hoặc learning-based AI theo độ phức tạp game.
- Tạo được prototype nhỏ có state, pathfinding, decision hoặc adaptation.

## 3. Nội dung roadmap
### Chủ đề

- Naive Bayes Classifier
- Decision Tree Learning
- Deep Learning
- Artificial Neural Network
- Reinforcement Learning

### Ứng dụng

- Bot học cách chơi game
- AI điều chỉnh độ khó
- Procedural content generation
- Player behavior prediction
- NPC adaptive behavior

## 4. Bài tập thực hành
- Tạo enemy AI mini với patrol, chase, attack hoặc flee.
- Biểu diễn decision logic bằng state machine, behavior tree hoặc flowchart.
- Test ít nhất ba tình huống để xem AI có hành vi mong muốn không.

## 5. Artifact nên tạo
- AI behavior prototype
- State machine/behavior tree diagram
- Test cases

## 6. Câu hỏi tự kiểm tra
- Tôi có thể giải thích **Machine Learning cho Game AI** bằng ví dụ trong một game cụ thể không?
- Chủ đề này ảnh hưởng đến gameplay, engine, graphics, physics, AI hay build pipeline?
- Demo hoặc ghi chú nào từ bài này có thể đưa vào portfolio game developer?

## 7. Tổng kết
**Machine Learning cho Game AI** nên được học bằng cách làm prototype, quan sát lỗi và ghi lại trade-off kỹ thuật. Hãy biến bài học thành một demo nhỏ hoặc README rõ ràng.
