# Project cuối khóa

Project cuối khóa là một game nhỏ hoàn chỉnh, đủ polish để đưa vào portfolio hoặc itch.io/Steam demo/mobile test.

## Yêu cầu tối thiểu

- Có gameplay loop rõ ràng.
- Có main menu hoặc màn hình bắt đầu.
- Có input, feedback, sound effect hoặc visual feedback cơ bản.
- Có ít nhất một hệ thống kỹ thuật nổi bật: physics, AI, shader, save/load, level system hoặc combat.
- Có bản build chạy được và README hướng dẫn chơi.

## Gợi ý thể loại

- 2D action platformer
- Top-down RPG
- Puzzle game
- FPS prototype
- Mobile casual game

## Deliverables

- Playable build hoặc link itch.io/demo.
- README mô tả gameplay, controls, tech stack và cách build.
- Screenshot hoặc GIF gameplay.
- Technical notes về ít nhất 3 hệ thống đã làm.
- Danh sách bug/limitation và kế hoạch cải tiến.
