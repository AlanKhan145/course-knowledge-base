# AI Engineer: From Zero to Production

Structured notes from a comprehensive AI Engineering course covering the full stack — from LLM fundamentals to production deployment.

## Course Structure

### Phase 0 — Foundations
- Introduction & course overview
- Python essentials for AI: FastAPI, async programming, Pydantic

### Phase 1 — Core LLM
- How LLMs work: tokenization, attention, context windows
- Sampling parameters: temperature, top-p, top-k

### Phase 2 — Prompt Engineering
- Zero-shot, few-shot, role-based prompting
- Structured output & function calling
- Context engineering

### Phase 3 — Working with Models
- OpenAI and Anthropic APIs
- Open-source models (Ollama, HuggingFace)

### Phase 4 — Embeddings & Vector Databases
- Embedding models and similarity search
- Vector DBs: ChromaDB, Pinecone

### Phase 5 — RAG (Retrieval-Augmented Generation)
- RAG fundamentals: chunking, retrieval, generation
- Advanced RAG: reranking, hybrid search, evaluation

### Phase 6 — AI Agents
- Agent architecture: planning, tool use, memory
- Multi-agent systems and orchestration

### Phase 7 — Model Context Protocol (MCP)
- MCP overview and integration patterns

### Phase 8 — AI Safety & Security
- Prompt injection prevention
- Safety guardrails and content moderation

### Phase 9 — Evaluation & Observability
- LLM evaluation frameworks
- Production monitoring and tracing

### Phase 10 — Capstone
- Multimodal AI (vision, audio)
- Final capstone project

## Files

Lesson notes are stored as `.txt` files under `AI-Engineer-Course/`, organized by phase and module number.

## Key Skills Covered

- Building production-ready LLM applications
- RAG pipelines from scratch
- Multi-agent orchestration
- Evaluating and monitoring LLMs in production
