# Welcome To This Course

## Overview
This opening lesson introduces the "AI Engineer from Zero to Production" course — a 20-week program designed to take intermediate developers from foundational AI concepts all the way to shipping production-grade AI systems. You will learn what to expect from the curriculum, how the course is structured across 10 phases, and what kind of AI Engineer you will become by the end.

## Key Points
- The course is organized into 10 progressive phases: Foundations, Core LLM, Prompt Engineering, Models, Embeddings and Vector DBs, RAG, Agents, MCP, Safety and Evaluation, and a final Capstone.
- Each phase builds on the previous — do not skip phases; concepts like token budgets and context windows introduced early are prerequisites for RAG and Agents later.
- The primary language is Python 3.11+; you should be comfortable with functions, classes, async/await, and virtual environments before starting.
- Every module contains a hands-on Lab lesson — these labs are the most important part; passive video watching does not build production skills.
- The course targets the 2024/2025 AI engineering stack: OpenAI / Anthropic APIs, LangChain, LlamaIndex, ChromaDB/Pinecone, FastAPI, Docker, and cloud deployment on AWS or GCP.

## What This Course Covers (Checklist)
- [ ] Phase 0 — Foundations: What AI Engineers do, core vocabulary, Python tooling
- [ ] Phase 1 — Core LLM: How large language models work internally; sampling parameters
- [ ] Phase 2 — Prompt Engineering: System prompts, structured output, context engineering
- [ ] Phase 3 — Models: OpenAI, Anthropic, open-source models (Llama 3, Mistral)
- [ ] Phase 4 — Embeddings and Vector DBs: Semantic search, ChromaDB, Pinecone
- [ ] Phase 5 — RAG: Retrieval-Augmented Generation from scratch to production
- [ ] Phase 6 — Agents: Tool use, planning, multi-agent architectures
- [ ] Phase 7 — MCP: Model Context Protocol for standardized tool integration
- [ ] Phase 8 — Safety and Security: Prompt injection, jailbreaks, guardrails
- [ ] Phase 9 — Evaluation and Observability: LLM-as-judge, tracing, monitoring
- [ ] Phase 10 — Multimodal AI and Final Capstone: Vision models, audio, end-to-end project

## Notes
This is a Module Introduction lesson — no code is required here, but do set up your environment before moving to Lesson 002. You will need Python 3.11+, a virtual environment manager (venv or conda), and API keys for OpenAI and/or Anthropic. Store keys in a .env file and never commit them to source control. Students with a data science background should pay particular attention to the production and deployment lessons — those sections cover skills that are rarely taught in ML courses.

## Summary
By the end of this course you will be able to design, build, and deploy production AI systems — not just call an API, but architect reliable, observable, and safe applications that deliver real business value.
