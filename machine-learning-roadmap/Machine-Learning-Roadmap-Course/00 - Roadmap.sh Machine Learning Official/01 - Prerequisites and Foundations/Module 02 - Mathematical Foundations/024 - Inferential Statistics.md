## 3. Khai niem chinh

- Inferential Statistics is part of the Statistics topic in the Machine Learning roadmap.
- Use statistics to understand data quality, variation, confidence and model errors.
- Look at distributions and segments before trusting a single aggregate metric.
- Learn it by connecting the definition, the dataset assumption, the model behavior and the evaluation impact.
- An ML Engineer should know where this concept appears in an end-to-end training workflow.
