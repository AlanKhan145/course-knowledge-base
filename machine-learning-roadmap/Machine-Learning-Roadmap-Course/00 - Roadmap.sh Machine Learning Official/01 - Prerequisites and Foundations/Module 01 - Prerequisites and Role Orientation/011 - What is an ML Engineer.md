## 1. Tom tat

Bai nay giai thich **What is an ML Engineer?** trong boi canh Machine Learning Engineer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong quy trinh ML, lien quan den data, feature, model, training, evaluation, deployment hoac portfolio nhu the nao.

## 2. Muc tieu hoc tap

- Giai thich duoc What is an ML Engineer? bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den model quality, generalization, interpretability, cost hoac production risk nao.
- Ap dung vao mot artifact nho: notebook, script, metric table, chart, diagram, model report hoac README.

## 3. Khai niem chinh

- An ML Engineer builds, trains, evaluates, optimizes and deploys machine learning models into real products.
- The role connects data engineering, modeling, software engineering, evaluation and production monitoring.
- A model is not finished when it trains; it must be reliable on unseen data and useful in a real workflow.
