## 3. Khai niem chinh

- Reinforcement learning trains an agent through interaction with an environment and reward feedback.
- The agent learns a policy that maps states to actions.
- Reward design and exploration strategy strongly shape behavior.

## 4. Thuc hanh

2. Define a tiny reward function for a toy problem.
3. Explain what behavior the reward encourages and what it may accidentally encourage.
