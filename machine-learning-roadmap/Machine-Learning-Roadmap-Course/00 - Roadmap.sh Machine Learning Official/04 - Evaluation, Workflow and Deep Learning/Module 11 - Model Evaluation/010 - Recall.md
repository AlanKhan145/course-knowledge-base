## 3. Khai niem chinh

- Recall answers: among true positives, how many did the model find?
- Use it when false negatives are costly.
- Examples include fraud missed by the model or sick patients marked healthy.

## 4. Thuc hanh

1. Calculate the metric or validation result on a small prediction table.
2. Explain when this metric is helpful and when it is misleading.
3. Connect the result to a decision: keep, tune, reject or investigate the model.

## 5. Bai tap
