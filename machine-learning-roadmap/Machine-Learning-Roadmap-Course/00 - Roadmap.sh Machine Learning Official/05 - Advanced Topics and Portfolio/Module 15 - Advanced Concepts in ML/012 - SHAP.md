## 1. Tom tat

Bai nay giai thich **SHAP** trong boi canh Machine Learning Engineer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong quy trinh ML, lien quan den data, feature, model, training, evaluation, deployment hoac portfolio nhu the nao.

## 2. Muc tieu hoc tap

- Giai thich duoc SHAP bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den model quality, generalization, interpretability, cost hoac production risk nao.
- Ap dung vao mot artifact nho: notebook, script, metric table, chart, diagram, model report hoac README.

## 3. Khai niem chinh

- SHAP estimates how features contribute to individual model predictions.
- It helps explain model behavior to stakeholders and debug surprising predictions.
- Explanations should be validated against domain knowledge and known data limitations.

## 4. Thuc hanh

1. Write a 5-line note explaining SHAP.
2. Add one ML example or counterexample.
3. Create a tiny artifact: notebook cell, diagram, checklist, metric table or README note.

## 5. Bai tap

Explain the concept, implement a small example and write how it affects model quality or reliability.
