## 1. Tom tat

Bai nay giai thich **setup-node action** trong boi canh QA Engineer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong quy trinh QA, manual testing, automation, API, non-functional testing, reporting, monitoring hoac CI/CD.

## 2. Muc tieu hoc tap

- Giai thich duoc setup-node action bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den risk, coverage, defect detection, release confidence hoac production quality nao.
- Ap dung vao mot artifact nho: test case, checklist, bug report, API collection, automation test, report, dashboard note hoac README.

## 3. Khai niem chinh

- setup-node action is part of the GitHub Actions for QA topic in the QA Engineer roadmap.
- Learn it by connecting the concept to risk, evidence, user impact and release confidence.
- A QA Engineer should know how to turn this topic into a test artifact or investigation note.

## 4. Thuc hanh

1. Run or write the command/config for the workflow.
2. Explain what output proves success.
3. Add a failure investigation note.

## 5. Bai tap

Explain the concept, create a small QA artifact and write how it affects release confidence.
