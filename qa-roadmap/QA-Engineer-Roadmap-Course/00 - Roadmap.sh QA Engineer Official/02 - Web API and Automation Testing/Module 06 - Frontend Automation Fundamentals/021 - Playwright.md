## 1. Tom tat

Bai nay giai thich **Playwright** trong boi canh QA Engineer Roadmap 2026. Sau bai hoc, ban nen biet topic nay nam o dau trong quy trinh QA, manual testing, automation, API, non-functional testing, reporting, monitoring hoac CI/CD.

## 2. Muc tieu hoc tap

- Giai thich duoc Playwright bang ngon ngu cua ban.
- Nhan biet topic nay anh huong den risk, coverage, defect detection, release confidence hoac production quality nao.
- Ap dung vao mot artifact nho: test case, checklist, bug report, API collection, automation test, report, dashboard note hoac README.

## 3. Khai niem chinh

- Playwright is a modern browser automation framework for end-to-end testing.
- It supports reliable locators, tracing, screenshots, videos and multi-browser execution.
- A practical QA portfolio can use Playwright for login, search, checkout and regression flows.

## 4. Thuc hanh

1. Automate a tiny browser flow or write framework-specific pseudocode.
2. Use a stable locator and meaningful assertion.
3. Record how the test reports failure.

## 5. Bai tap

Automate or pseudocode one UI flow with a stable locator and clear assertion.
