# Capstone Projects

Các project cuối khóa giúp gom developer journey, communication, technical demo, community, content, onboarding và metrics thành portfolio DevRel.

## Project 01 - Developer Journey and DX Audit

File chi tiết: [Developer Journey and DX Audit](99 - Portfolio Capstone Projects/Project 01 - Developer Journey and DX Audit.md)

Bạn phụ trách một API/SDK mới và cần đánh giá hành trình developer từ lúc biết tới sản phẩm đến khi build thành công demo đầu tiên.

Deliverable chính:
- Developer journey map
- DX audit checklist
- Top 10 friction points
- Prioritized improvement plan
- Time-to-first-success metric

## Project 02 - Technical Content Campaign

File chi tiết: [Technical Content Campaign](99 - Portfolio Capstone Projects/Project 02 - Technical Content Campaign.md)

Bạn cần tạo một chiến dịch nội dung 4 tuần để giúp developer hiểu và thử một tính năng mới.

Deliverable chính:
- Campaign brief
- Content calendar
- Technical blog/tutorial
- Video or livestream script
- Social repurpose pack

## Project 03 - Community Launch Plan

File chi tiết: [Community Launch Plan](99 - Portfolio Capstone Projects/Project 03 - Community Launch Plan.md)

Deliverable chính:

- Community guidelines
- Moderation playbook
- First event plan
- Recognition program

## Project 04 - Developer Onboarding Kit

File chi tiết: [Developer Onboarding Kit](99 - Portfolio Capstone Projects/Project 04 - Developer Onboarding Kit.md)

## Project 05 - DevRel Metrics and Reporting System

File chi tiết: [DevRel Metrics and Reporting System](99 - Portfolio Capstone Projects/Project 05 - DevRel Metrics and Reporting System.md)
