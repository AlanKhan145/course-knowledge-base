# Module 04 - Community Building

## Mục tiêu module
Biết xây, điều phối và phát triển cộng đồng developer an toàn, có giá trị và có nhịp hoạt động.

## Bài học
- [001 - Building a Community](001 - Building a Community.md)
- [002 - Community Guidelines](002 - Community Guidelines.md)
- [003 - Community Management](003 - Community Management.md)
- [004 - Event Management](004 - Event Management.md)

## Thực hành trọng tâm
Thiết kế community space cho một sản phẩm kỹ thuật: audience, rules, moderation, event đầu tiên.

## Output sau module
- Community launch plan, guidelines and moderation playbook
- Một decision note hoặc content/support/community plan có thể chia sẻ với team
- Danh sách friction, insight hoặc feedback cần gửi lại product/engineering
