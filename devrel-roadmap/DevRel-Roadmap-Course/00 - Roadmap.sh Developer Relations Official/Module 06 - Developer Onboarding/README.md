# Module 06 - Developer Onboarding

## Mục tiêu module
Biết thiết kế onboarding giúp developer đạt time-to-first-success nhanh hơn.

## Bài học
- [001 - Documentation](001 - Documentation.md)
- [002 - Sample Projects](002 - Sample Projects.md)

## Thực hành trọng tâm
Audit onboarding của một API/SDK: quickstart, docs, sample app, FAQ và support channel.

## Output sau module
- Developer onboarding pack: docs, sample project, support flow and office hour plan
- Một decision note hoặc content/support/community plan có thể chia sẻ với team
- Danh sách friction, insight hoặc feedback cần gửi lại product/engineering
