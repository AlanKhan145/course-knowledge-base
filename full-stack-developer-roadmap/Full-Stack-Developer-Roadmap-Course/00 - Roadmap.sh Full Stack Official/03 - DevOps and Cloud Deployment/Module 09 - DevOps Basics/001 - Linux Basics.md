## 1. Tom tat

Bai nay giai thich **Linux Basics** trong boi canh full-stack hien dai. Sau bai hoc, ban nen biet no nam o dau trong hanh trinh tu frontend den backend, database va deployment.

## 2. Muc tieu hoc tap

- Giai thich duoc Linux Basics bang ngon ngu cua ban.
- Nhan biet kien thuc nay nam o frontend, backend, database, cloud hay workflow.
- Ap dung vao mot vi du nho trong full-stack project.

## 3. Khai niem chinh

- Linux la moi truong pho bien de deploy web app va service backend.
- Can biet shell, file permission, process, port, service va log co ban.
- Debug production thuong bat dau tu ssh, env, disk, memory, network va log.

## 4. Vi du / Demo

```bash
systemctl status app
journalctl -u app --since "10 minutes ago"
ss -ltnp
```

## 5. Bai tap thuc hanh

- Ve so do luong request tu domain den app va cac service lien quan.
- Tao runbook ngan: lenh kiem tra, loi thuong gap va cach rollback.
- Neu co the, thuc hanh tren sandbox cloud hoac server test.

## 6. Loi thuong gap

- Lam theo tutorial ma khong hieu secret, permission va boundary.
- Khong test failure case nhu token het han, service down hoac deploy loi.
- Thieu log/runbook nen khi loi khong biet bat dau debug o dau.

## 7. Checklist hoan thanh

- Toi co the giai thich **Linux Basics** trong 1-2 phut.
- Toi co mot vi du nho hoac ghi chu thuc hanh cho bai nay.
- Toi biet bai nay ket noi voi phan nao cua app full-stack.
- Toi da ghi lai it nhat mot cau hoi can tim hieu sau neu con mo ho.

## 8. Outcome lien quan

Deploy a full-stack app to cloud infrastructure

## 9. Tong ket

**Linux Basics** la mot moc trong lo trinh full-stack. Hay bien no thanh mot vi du nho, mot checklist debug, hoac mot project mini de kien thuc co cho bam.
