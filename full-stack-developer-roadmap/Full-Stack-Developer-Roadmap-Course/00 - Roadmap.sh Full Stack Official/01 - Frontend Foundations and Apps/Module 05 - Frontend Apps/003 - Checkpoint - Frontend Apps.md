## 1. Tom tat

Bai checkpoint nay bien module **Frontend Apps** thanh san pham nho co the demo. Muc tieu la chung minh ban khong chi biet ly thuyet ma con ghep duoc cac phan cua mot app that.

## 2. Muc tieu hoc tap

- Chuyen kien thuc vua hoc thanh artifact co the demo.
- Dat tieu chi hoan thanh ro truoc khi lam.
- Ghi lai cach chay, cach test va loi gap phai.

## 3. Khai niem chinh

- Checkpoint - Frontend Apps la moc thuc hanh de kiem tra module Frontend Apps.
- Checkpoint nen tao artifact co the demo, review va dua vao portfolio.
- Dung tieu chi done ro rang: chay duoc local, co README, co test/checklist va co ghi chu loi gap phai.

## 4. Vi du / Demo

```text
Goal -> Build -> Test -> Document -> Demo
Example: complete app -> auth + CRUD + cache -> smoke test -> README -> short walkthrough
```

## 5. Bai tap thuc hanh

- Xay mini React app co danh sach, form them/sua va loading/error state.
- Tach component co y nghia va dung Tailwind de style co he thong.
- Ket noi voi mock API hoac local JSON.

## 6. Loi thuong gap

- Bat dau project qua lon lam mat dong luc.
- Khong viet README/cach chay nen project kho review.
- Chi lam happy path ma bo qua error, validation va deployment note.

## 7. Checklist hoan thanh

- Toi co the giai thich **Checkpoint - Frontend Apps** trong 1-2 phut.
- Toi co mot vi du nho hoac ghi chu thuc hanh cho bai nay.
- Toi biet bai nay ket noi voi phan nao cua app full-stack.
- Toi da ghi lai it nhat mot cau hoi can tim hieu sau neu con mo ho.

## 8. Outcome lien quan

Build component-based frontend applications. The roadmap notes these can be skipped and revisited after backend

## 9. Tong ket

**Checkpoint - Frontend Apps** la mot moc trong lo trinh full-stack. Hay bien no thanh mot vi du nho, mot checklist debug, hoac mot project mini de kien thuc co cho bam.
