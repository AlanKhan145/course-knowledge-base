## 3. Khai niem chinh

- Alerting is part of the Observability topic in the Data Engineer roadmap.
- Observe systems through metrics, logs and traces tied to clear service-level expectations.
- Dashboards should answer operational questions, not merely display many charts.
- Study it by connecting the definition, the production use case and the failure modes.
- A Data Engineer should be able to explain where this fits in a pipeline and how to validate it.
