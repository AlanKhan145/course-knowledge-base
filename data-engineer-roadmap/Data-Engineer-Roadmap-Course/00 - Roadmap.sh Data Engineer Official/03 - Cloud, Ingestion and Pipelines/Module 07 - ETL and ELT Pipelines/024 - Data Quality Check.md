## 3. Khai niem chinh

- Data Quality Check is part of the Pipeline Tools topic in the Data Engineer roadmap.
- Study it by connecting the definition, the production use case and the failure modes.
- A Data Engineer should be able to explain where this fits in a pipeline and how to validate it.
- Turn the concept into a repeatable check that can fail a pipeline before bad data is served.
