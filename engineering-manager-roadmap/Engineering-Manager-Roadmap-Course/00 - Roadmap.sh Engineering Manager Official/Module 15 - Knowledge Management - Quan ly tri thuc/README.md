# Module 15 - Knowledge Management - Quản lý tri thức

## Mục tiêu module

## Nội dung

## Thực hành trọng tâm
Thiết kế knowledge management workflow: tài liệu nào phải có, owner là ai, review cadence thế nào.

## Output sau module

- Một decision note hoặc checklist có thể dùng lại
- Danh sách rủi ro, câu hỏi mở hoặc việc cần follow-up
