## Thực hành trọng tâm
Chọn một capstone project và hoàn thành như một portfolio artifact có thể review.

## Output sau module
- Engineering manager portfolio package
- Một decision note hoặc checklist có thể dùng lại
- Danh sách rủi ro, câu hỏi mở hoặc việc cần follow-up

## Checklist hoàn thành
- [ ] Đọc toàn bộ bài học/project trong module

- [ ] Liên hệ nội dung với một team, hệ thống hoặc dự án thật
- [ ] Ghi lại 3 điều sẽ áp dụng trong vai trò Engineering Manager
