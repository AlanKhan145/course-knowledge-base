# 005 - Related Roadmaps nên học kèm

**Module:** Module 17 - Capstone Project - Dự án cuối khóa  
**Loại nội dung:** Learning path  
**Thời lượng gợi ý:** 20-30 phút

---

## 1. Tóm tắt
Engineering Manager cần đủ sâu về kỹ thuật để dẫn dắt architecture, system design, backend, DevOps và technical decision-making. Các roadmap liên quan giúp bổ sung năng lực nền.

## 2. Roadmap nên học kèm
- Software Architect
- System Design
- Software Design & Architecture
- Backend Development
- DevOps Engineering

## 3. Cách dùng
- Chọn 1-2 roadmap gần nhất với điểm yếu hiện tại của bạn.
- Không cần học hết trước khi làm EM; hãy học theo vấn đề team đang gặp.
- Biến mỗi roadmap phụ thành checklist năng lực: biết gì, thiếu gì, artifact nào chứng minh được.

## 4. Artifact nên tạo
- Personal learning plan 30 ngày
- Skill gap matrix cho vai trò EM
- Danh sách chủ đề cần hỏi mentor hoặc senior engineer
