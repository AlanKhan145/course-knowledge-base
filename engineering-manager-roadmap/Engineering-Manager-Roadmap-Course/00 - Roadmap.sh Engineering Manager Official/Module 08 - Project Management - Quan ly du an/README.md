## Thực hành trọng tâm
Lập kế hoạch delivery cho một dự án 6-8 tuần: resource, sprint, milestone, risk, dependency và scope.

## Output sau module
- Delivery plan, risk register and dependency map
- Một decision note hoặc checklist có thể dùng lại
- Danh sách rủi ro, câu hỏi mở hoặc việc cần follow-up
