# Module 16 - Change Management - Quản lý thay đổi

## Mục tiêu module
Biết dẫn dắt technical, organizational và team change mà vẫn giữ alignment.

## Nội dung
- [001 - Technical Change](001 - Technical Change.md)
- [002 - Organizational Change](002 - Organizational Change.md)
- [003 - Team Change](003 - Team Change.md)

## Thực hành trọng tâm
Lập change plan cho migration, reorg hoặc process change: impact, stakeholder, communication và resistance.

## Output sau module
- Change management plan
- Một decision note hoặc checklist có thể dùng lại
- Danh sách rủi ro, câu hỏi mở hoặc việc cần follow-up

## Checklist hoàn thành
- [ ] Đọc toàn bộ bài học/project trong module
- [ ] Tạo ít nhất một artifact thực hành
- [ ] Liên hệ nội dung với một team, hệ thống hoặc dự án thật
- [ ] Ghi lại 3 điều sẽ áp dụng trong vai trò Engineering Manager
